■ string

끈을 일컫는 가장 일반적인 말

​

a box tied with string

끈으로 묶인 박스

​

■ twine

실을 꼬아서 만든 끈. 특히 밧줄이나 포장용 끈을 말한다.

​

Why don't you tie up the bos with this twine?

이 끈으로 상자를 묶지 그래?

​

■ cord

string보다 약간 더 두껍고 단단한 끈

​

Tie it with a cord tightly.

끈으로 단단히 묶으세요

​

■ strap

물건에 달려 있는 띠 모양의 끈으로 어떤 것을 운반하거나 잡을 때 쓰는 것

​

camera strap (카메라의 어깨끈) / chinstrap (헬멧 등의 끈) / shoulder strap (수영복 등의 어깨끈)
