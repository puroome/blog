build / construct / assemble / erect / set up / put up / put together
▷ 이 단어들은 모두, 특히 여러 가지 다른 부분들을 함께 조합하여, 무엇을 만드는 것을 나타낸다.

■ build(특히 건물을) 짓다[건축하다]:
a house built of stone
돌로 지은[석조] 주택
They’re going to build on the site of the old power station.
그들은 구 발전소 부지에 건축을 할 예정이다.

■ construct [흔히 수동태로] (비교적 격식) (도로ㆍ건물ㆍ기계를) 건설[설치]하다
When was the bridge constructed?
그 다리는 언제 건설되었죠?

■ assemble (비교적 격식) (가구나 기계를) 조립하다:
The cupboard is easy to assemble.
그 찬장은 조립이 쉽다.

■ erect (격식) 건축[설치]하다; (똑바로) 세우다:
Police had to erect barriers to keep crowds back.
사람들이 다가오지 못하도록 경찰이 차단물을 세워야 했다.

■ set sth up (무엇을 어디에) 세우다[설치하다]:
We decided to set up camp for the night.
우리는 천막을 치고 그날 밤을 보내기로 했다.

■ put sth up (무엇을 어디에) 세우다[건립하다]:
They’re putting up new hotels in order to boost tourism in the area.
그 지역의 관광 산업을 활성화하기 위해 새로 호텔들을 건립하고 있다.

set sth up or put sth up?
◇set sth up은 영구적인 건물에 대해서는 쓰지 않는다:
They’re setting up new hotels.

◇‘캠프를 만들다’는 set up camp로, ‘텐트를 치다’는 put up a tent로 표현한다.

◇임시적으로 만드는 다른 건축물에 대해서는 어느 어구나 쓸 수 있다:
to set up / put up a fence / barrier / shelter
울타리 / 차단물 / 대피소를 세우다.

■ put sth together (여러 부분들을 모아) 만들다[조립하다]:
to put together a model plane
모형 비행기를 조립하다
