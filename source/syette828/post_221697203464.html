영어로 '공을 정확히 차다' (영어로 정확히,정확하게?) 

exact, accurate, correct 차이? 






이 포스팅을 다 보시면 한국말 '정확하게' 가 


얼마나 사용하기 쉽고 좋은 단어인지 알게 되실 겁니다. 


영어는 가끔 너무나 어렵습니다. ㅠㅠ







정확히, 정확하게는 부사로서 


사전에 찾아보면 exactly, accurately, correctly 가 


유사어로서 나옵니다. 


하지만 아무 상황에서 이 단어 중 하나를 골라 쓰는 것은 


옳지 않습니다. 


그래서 아래서 각 단어에 대해서 정확한 뜻을 알아보고


예문으로 공부할 예정입니다. 







1. Exact (Exactly) 






exact = 정확한 (형용사)


exactly = 정확하게, 정확히, 틀림없이, 꼭, 맞아 (부사)








exact 는 세 단어 중에 쉽게 자주 쓰이는 단어입니다. 


exact 는 '올바른, 맞은' 의 정확한 이란 뜻도 있지만 그 보다


'딱 맞는' 에 더욱 적합하다고 볼 수 있습니다. 








ex)
Exactly! = 그거야. (맞아, 그렇지, 정확해 등...) 



It's been exactly 3 years since I've studied Chinese. 

= 중국어 공부한지 정확히(딱) 3년 되었어. 



The exact distance is 100 kms. = 정확한 거리는 100 킬로미터야. 

* 올바른 거리라고 하면 해석이 어색하니 exact! 



I can't remember the exact date. = 정확한 날짜는 기억이 안나. 



Do you know exactly how many people came? 

= 너 정확히 몇 명 왔는지 알아?





 



2. Correct (Correctly)




correct = 정확한, 맞는, 옳은, 올바른  (형용사)


correctly = 정확하게, 올바르게, 맞게... (부사)


correct = 수정하다, 고치다, 올바르게 하다 (동사)









correct 는 '정확한' 이라는 뜻도 가능하지만 


'올바른, 맞은' 의 의미가 내포되어 있습니다. 


그렇기에 exact 와 correct 이 헷갈리면  


'올바른, 올바르게' 로 대체될 수 있다면


correct(ly) 를 사용하시면 됩니다.




ex)

You should put the correct number. 

= 넌 올바른 숫자를 넣어야(기입해야) 해. 



How do I say this word correctly? 

= 이 단어를 어떻게 정확히(올바르게) 말해? 



Can you correct my grammar? = 내 문법 좀 고쳐줄 수 있어? 

* 동사로 사용 됨. 



You should do it correctly. = 너 이거 정확히 해야 해. 

* 보통 ~ 를 제대로, 정확히 하라는 말을 할 때는 

  exact 가 아닌 correct 를 사용해야 함. 



The correct answer is this. 

=올바른 정답은 이거야. 








3. Accurate (Accurately) 



accurate = 정확한 (형용사)


accurately = 정확하게 (부사)


accuracy = 정확, 정확도 (명사)






accurate 는 우선 발음이 조금 어렵습니다. 


(애큐래잇) 처럼 발음 하셔야 합니다. 




영영사전에서 보셨듯이 accurate은 exact 와 correct 의 의미를 갖습니다. 


그래서 동의어로 사용가능하기도 하지만 


주로 수치나, 세밀함에 대한 정확성에 대해서 표현할 때 사용합니다.




ex)

I want to kick(strike) the ball accurately. 

= I want to kick(strike) the ball with accuracy. 

= 난 공을 정확하게 차고 싶어. 


* <with + 명사> 이기에 accuracy 를 넣습니다. 

  이 예문처럼 공을 올바르게.. 의 느낌도 아니고..

  딱 맞게.. 의 느낌도 아닌 '정확성' 이라는 의미로 

  표현하고 싶다면 accurate을 사용합니다.
