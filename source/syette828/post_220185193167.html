if 는 흔히 만약에~ 라면  으로 쓰이죠. 

 

하지만! ~인지 아닌지 로도 회화에서 자주 쓰입니다. 

 

그렇다면 어떻게 쓰이는지 증명해 보이겠습니다. ^^

 

 

 

 

 

If 와 같이 자주오는 주절을 먼저 알아봅시다. 

 

 

I want to see if +S + V  =  ~인지 보고싶다(알고싶다)

 

I want to know if +S + V  =  ~인지 알고싶다

 

I wonder if +S + V  = ~인지 궁금하다

 

I came to see if +S + V  = ~인지 궁금해서 왔다

 

.

.

.

 

 

예문)

 

I want to see if she likes it = 나는 그녀가 그것을 좋아하는지 보고싶다. 

 

I wanted to know if he was mad = 나는 그가 화났는지 알고 싶었다. 

 

I wonder if it is true = 나는 그것이 사실인지 궁금하다.

 

I came to see if I could see Mr.Kim = 김씨를 볼 수 있나해서 왔습니다. (김씨를 만나뵐 수 있을까 해서 찾아왔습니다.)

 

 

 

위의 4문장이 왜 중요한지 모르시는 분들도 있다고 생각합니다. '

 

그렇지만 회화에서 정~말 자주 쓰이는 표현입니다. 

 

마지막 예문의 경우를 중점적으로 설명할게요. 

 

우리가 흔히 어디가를 방문해서 방문목적을 이야기해야 할 경우가 많죠~~

 

예를 들어, 도서관에 가서 "~ 책 빌리려고 왔는데요. "

 

 

 

이 표현을 영어 초급자에게 물어보면 

 

I want to borrow some books. 이렇게 얘기합니다. 

 

이 경우는 틀렸다고 말할 수 없지만 상대가 듣기에 좀 어색하고 웬지 

 

어린이가 (저~하고 싶어요) 이런 느낌이 나요. 

 

저 ~ 하고 싶어요. 이러면 이상하잖아요 ㅋㅋ

 

﻿ 

 

중급레벨로 치자면 

 

I came to borrow some books.(책 빌리러 왔습니다) 이렇게 나옵니다. 

 

이 경우는 올바른 문장입니다. 하지만  중급이라고 

 

한 이유가 상대방이 듣기에 아주 매너있고 젠틀한 말투는 아니기 때문입니다. 

 

 

 

그렇다면 상급자! ^^ 가 되기 위해서는

 

I came to see If I could borrow some books here. 

 

이렇게 바꾸시면 "여기서 책을 빌릴 수 있을까 해서요~ " 라는 멋진 말이 됩니다. 

 

 

 

또 다른 예로!

 

교수님을 방문하러 가서 

 

I want to see Professor Kim. 이라고 하기 보다

 

I wonder If I can(could) see Professor Kim today. 이라고 말하면 훨~씬 듣기 좋습니다. 

 

 

여기서 can 과 could 에서도 could 는 덜 확신적인 느낌으로 더 정중한 느낌이 듭니다. 

 

왜냐면 "~할 수 있다면" 의 느낌이 나서 그래요. 

 

 

 

이제 I want to do~ 이란 표현애서 한 번 벗어나 보세요!!
