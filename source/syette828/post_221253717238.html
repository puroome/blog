문법을 단순히 외우는 것은 좋은 방법이 아닙니다. 


이해하신다면 자동으로 외워지면 다양하게 응용할 수 있게 됩니다. 


그렇기에 시간이 좀 걸리더라도 앞의 강의를 보지 않으신 분들은 


꼭 먼저 보시길 권해 드립니다. (34,35강) 







앞서 35강에서는 가정법 if 를 이용해서 현재와 미래상황을 가정하는 것을 배웠습니다. 


<조동사 + have pp>와  if 가정법 (과거상황가정) 이 자주 쓰이기 때문에


이 부분을 먼저 알려드리겠습니다. 








*가정법 과거상황 가정



해석 = 예전에 ~ 했다면, 했었다면 등..



가정법은 가정하고자 하는 상황의 시제보다 항상 한 시제 뒤로 더 가야합니다. 


그래야 현실인지 상상하는 것인지 파악할 수 있기 때문입니다. 


<문법적으로도 말이 되지 않기에 = 상상,가정>  이렇게 이해하시면 좋을 것 같네요. 



나는 공부했다 = I studied 


내가 공부했었더라면 = If I had studied (O), if I studied(X) 



* 중요팁! 

  문법적으로 had pp가 당연히 맞지만,


  구어체에서 원어민들도 종종 단순과거를(~ed,ate,did,studied..)사용하는 걸 들으실 겁니다.


  또한 사용하셔도 크게 문제되지 않습니다. ^^




ex) 

If I had eaten that = 내가 만약 그걸 먹었다면 (실제로 먹지 않음) 


If I had studied harder = 내가 만약 공부를 더 열심히 했더라면 

* 팁! 긍정문은 if I had = If I'd 로 줄일 수 있습니다. 

      부정문은 줄일 수 없고 I hadn't (had not)로 표현


If I hadn't gone there = 내가 만약 거길 가지 않았었다면 


If she hadn't seen me there = 만약 그녀가 날 거기서 보지 않았었다면 









이제 본격적으로 <조동사 + have pp > 를 살펴보겠습니다. 



1. would have pp 



would 는 will 과 비교되어지는 조동사 입니다. 


would는 will 보다 가능성이 낮은 느낌의 '~일텐데, 일것이다' 로 해석됩니다. 


would 가 have pp (과거= ~했다) 와 만나면 


would have pp = ~했을 거야(다) / ~했을텐데 (실제로 하지 않음) 



* 발음 

would have = would've (우드브) = woulda or would of (빠르게 발음)  처럼 발음됩니다. 



문어체에서는 무조건 would have pp형식으로 사용하시고 


구어체에서는 줄여진 형태로 발음하시는 걸 연습하세요. 




ex) 

If I had seen you at the party, I would've said 'hi'. 

/ 내가 파티에서 널 봤다면, 와서 인사했을거야. (실제로 보지도 못했고 인사도 못했음)


If I'd (=I had) studied harder, I would've passed this exam. 

/ 내가 공부를 더 열심히 했다면, 이 시험 패스했을텐데. 


If 손흥민 had scored that, team Korea would've won that game. 

/ 만약 손흥민이 그걸 넣었다면, 한국팀이 이겼을거야. (손흥민님 팬입니다. ^^ 예문을 위해서 ㅠ) 


I would've gone to the wedding but I couldn't skip work. (but because of my work...) 

/결혼식장에 갔을거야 일만 아니었다면 (근데 일을 빼먹을 수가 없어서, 쨀 수가 없어서 ) 


If she'd asked me in a nicer way, I would've said 'yes'. 

/그녀가 좀 더 친절하게 물어봤다면 좋다고(괜찮다고) 말했을 거야. 


I would've done it differently. / 나라면 다르게 했을 거야. 

* 어떤 상황인지 말 안해도 명백하다면 if절 생략해도 됩니다. 


Oh man, I wouldn't have done that. 

/ 오 이런, 나라면 안 그랬을 거야. 


I would've said 'sorry' in that situation. 

/ 그런 상황에서 난 미안하다고 했을거야. 







 





2. could have pp 



can = 할 수 있다 라는 (능력) 을 표현합니다. 


could 는 can 과 비교되는 조동사로  =~할 수 있을텐데 (가능성 낮출 때) 로 쓰입니다. 




could have pp = ~할 수도 있었어(다),  / ~할 수 있었을텐데 


즉, ~를 할 수도 있었지만 (기회나 능력이 있었지만) 결국 하지 못하거나 안한 일에 대해 표현! 

 


발음은 could've (쿠르브) = could of (빠르게) coulda (쿠더,쿠러) 처럼 발음 하시면 됩니다.




ex) 

I could have eaten more if I had had more time. 

/ 시간만 더 있었다면 더 먹을 수도 있었어. 

* had가 두번 들어간건 처음 had는 had pp의 had, 두 번째 had 는 have 의 pp 


I could have gone there, but I chose not to. 

/ 난 거기 갈 수도 있었지만 안가기로 결정했어. 


I could've got into any university, if I had applied. 

/ 내가 지원만 했다면 어느 대학도 들어갈 수 있었어. 


We could've visited more places in Japan, if one of my friends hadn't been sick. 

/ 만약 내 친구중 한 명이 아프지만 않았다면, 우린 일본에서 더 많은 곳을 볼 수 있었어. 


You totally could have asked her number! Why did you let her go?! 

/ 너 완전 그 여자애 번호 물어볼 수 있었는데! 왜 그냥 가게 냅뒀어?! 


You could've done it better if you had paid more attention (to it).   

/ 너가 더 집중했었다면 더 잘 할 수도 있었어. 


It could've been a lot worse. 

/ 훨씬 더 상황이 안 좋을 수도 있었어. (하지만 다행이도 더 나쁘진 않았다는 뜻) 


If I had got just 5 more points(scores), I could have even gone to 연세university. 

/ 내가 5점만 더 받았다면, 난 연세대학도 갈 수 있었을텐데 (있었을거야) 


It could've happened to anyone. 

/ 그건 누구에게도 일어날 수 있었어. 

* It can happen to anyone = 그건 누구에게나 일어날 수 있어. 











3. should have pp 



should 는 '~하는 것이 좋다, ~해야한다' 의 의미를 갖는 조동사입니다. 


should have pp = ~했어야 했다, ~할 걸 그랬어 


즉, 했어야 했는데 하지 못한 일에 대한 후회, 아쉬움을 표현합니다. 




발음은 should've (슈드브) or should of (빠르게) or shoulda (슈다,슈더) 




ex) 

You should've said that earlier. / 너 그걸 더 일찍 말했어야지. 

(즉, 상대방이 일찍 말하지 않아서 아쉬움)


We should have left earlier to get tickets. 

/ 우린 티켓을 사기 위해 좀 더 일찍 출발 했어야 했다. (결국 못삼) 


You should've told the truth. / 넌 진실을 말했어야 했다. 


We shouldn't have drunk so much. / 우리 술을 너무 마시면 안됐어. (마시지 말걸 그랬어)  


It's freezing cold here in Canada, I should've brought more winter clothes. 

/ 여기 캐나다 엄청 추워, 나 겨울 옷 좀 더 가져올 걸 그랬어. 


I shouldn't have bought these shoes, they are a lot cheaper online. 

/ 나 이 신발 사지 말았어야 했어. 온라인에서 훨씬 싸. ㅠㅠ


These are 3 things I should have done in college.  

/ 이것들이 3가지 내가 대학다닐 때 했어야 했던 후회되는 일이야. 


You should've told me you were coming because I can't go out now. 

/ 여기 오는 중이라고 말했어야지. 나 지금 못 나간단 말야. 


Hey!, You should've scored there. Why did you pass the ball?!!

/ 야! 거기서 슛했어야지 (골 넣었어야지). 왜 패스했어?! 


I should've studied English more, cuz I'm going to California for the business trip. 

/ 나 영어 공부 좀 더 할 걸 그랬어, 캘리포니아로 출장 가게 되었어. 









 




4. might have pp 



might = ~일지도 모른다 (약한 추측, 자신 없는 추측) 


might have pp = ~이었을지도 모른다 , ~했을지도 모른다, 아마 ~ 했나봐 



발음은 might've (마이르브) or  might of (빠르게)  or mightda (마이르,마이러)  




ex) 

He might have forgotten my name cuz he never calls me by name. 

/ 그는 내 이름을 까먹었는지도 몰라 왜냐면 내 이름을 전혀 안 불러. 


If you had been there on time, you might have seen her. 

/ 네가 만약 제 시간에 거기 왔다면, 그녀를 볼 수 있었을지도 몰라. 


It that hadn't happened, this might've happened instead. 

/ 만약 그게 일어나지 않았었다면, 대신 이 일이 일어났을지도 몰라. 


You're so lucky. If you had stayed longer someone might have killed you. 

/ 너 운 엄청 좋은거야. 더 오래 거기 있었다면 누군가가 널 죽였을지도 몰라. 


This is what might have happened to you. 

/ 이게 너한테 일어났을지도 몰랐던 거야. 


best 5 movies you might have missed in 2017 / 당신이 놓쳤을지도 모르는 2017년 최고의 영화 5










5. must have pp 


must = '틀림없이 ~이다' (강한 추측) 


must have pp  = 틀림없이 ~이었을거야 / 분명 ~ 했나봐 / ~했던 모양이야 

(강한 가능성 추측) 



발음 = must've (머스뜨브) or must of (빠르게) or mustda (머스따, 머스떠) 




ex) 

You must have been busy because you never miss that meeting.

/ 너 바빴었나봐? 그 모임 빠지는 일 없잖아. 

* 항상 오던 사람이 지난 모임에 오지 않아서 분명 바빴었구나라고 하는 표현


You really ate this all? You must've been hungry. 

/ 너 진짜 이거 다 먹었어?? 완전 배고팠었구나. (배고팠던 모양이구나) 


He must have taken the wrong exit. / 걔 분명 다른 길로 빠졌을거야. 


You must have freaked out! / 너 엄청 놀랐었겠다. (출처: 능률교육) 


Her car is gone, so she must've left already.  (출처: 능률교육)

그녀의 자동차가 없는걸 보니 이미 떠난게 분명하네.(이미 떠난 모양이네) 








마지막으로 비슷한 단어로 각각의 표현을 비교해 드릴게요. 



I could've eaten more if I had had more time. 

/ 시간이 더 있었다면 더 먹을 수도 있었어. (하지만 먹지 못 함) (그럴 기회, 능력 있었음)  


I would've eaten more if I had known I wasn't paying this. 

/ 만약 내가 사는 게 아니라는 걸 알았었다면 더 먹었을 거야. (과거에 will 일것이다, 했을거야) 


You can't even walk? You must've eaten a lot. 

/ 걷지도 못하겠어? 엄청 먹은 모양이네.(엄청 많이 먹었나보군) (강한 추측)


A: Who ate my food? / 누가 내 음식 먹었어? 

B: He might have eaten that because he seems nervous. 

   / 쟤가 먹었을지도 몰라 왜냐면 뭔가 초조해 보여. (약한 추측)


I should have eaten more because I'm already hungry. 

/ 나 좀 더 먹었어야 했어 왜냐면 벌써 배고파. (후회)
