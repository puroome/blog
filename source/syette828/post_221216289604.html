영어로 '얼마나 아팠을까, 얼마나 힘들었을까, 얼마나~일까,할까' 표현하기 


 





얼마나 힘들었을까..??  


이 표현을 배우기에 앞서 must 와  must have pp 형태를 알아야 합니다. 


must 는  '~을 해야한다' 말고도 '틀림없이 ~이다' 의 뜻도 있습니다. 




ex) 

She must be sick. / 그녀는 분명 아플거야. (강한 확신) 


He must be somewhere here. / 그는 분명 여기 어딘가에 있을거야. 


He must work here. / 그는 틀림없이 여기서 일 할거야. 






must + have pp 는 '틀림없이 ~였을거야' / '분명이 ~였을거야' .. 로 해석되며


과거의 어떤 일에 대해서 강한 추측을 할 때 씁니다. 



ex) 

He must have (must've) said something. / 그는 틀림없이 무언가를 말했을거야. 








영어로 '얼마나 힘들었을까?'  


It(something) must have been (so) difficult (hard, rough) + for + 사람 


으로 표현할 수 있습니다. 



또한 I can't imagine how ~~  형태로도 가능합니다. 



ex)

Oh, man. It must've been very hard for me to go through. 

/ 오 이런, 그건 그가 겪기에 정말 힘들었을거야. (얼마나 그한테 겪기 힘들었을까...) 

 * go through = 경험하다, 겪다 



I can't imagine how difficult it was for me to go through that. 

/ 그런 일을 겪다니 얼마나 힘들었을지 상상이 안가. 










영어로 얼마나 아팠을까 / 아플까 ? 



얼마나 아팠을까...(과거사건묘사) 

It must have been so painful. 

* painful = 고통스러운 




얼마나 아플까...(현재 일어난 일)

It must be so painful. 

Oh, that's so painful. 

Oh, that hurts. 

Ouch. / 아이구야. 아야. (갑자기 아파서 내는 소리 or 상대방의 아픔이 느껴질 때)  









영어로 얼마나 무서웠을까 / 무서울까? 



얼마나 무서웠을까.. (과거) 

It must've been so scary (for him). 




얼마나 무서울까.. / 완전 무섭겠다 (현재) 

How scary!  

It must be so scary. 

That's so scary. 




얼마나 무서울까 (미래) 

I don't know how scary it would be.(is going to be) / 얼마나 무서울지 모르겠어요. 

I can't imagine how scary ~~  . / 얼마나 무서울지 상상이 안 가. 

It must be so scary.  * 현재, 미래 둘 다 가능합니다!  미래 강한 추측! 

It would be so scary. / 엄청 무서울거야.  미래 덜 강한 추측! 








이제 감이 잡히시나요?? 


얼마나 ~일까, 했을까 와 완벽하게 그리고 똑같이 해석되는 영어표현은 없습니다. 


how,  I can't imagine, must have pp... 등으로 표현하면 됩니다.
