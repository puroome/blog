영어로 "오랜만에 ~했어/5년만에 ~했어/~한지 오래되었어 등.." 

in a long time / for a long time 차이 


 



영어로 '오랜만에' 라는 표현은 한국말처럼 쉽게 표현하기 힘듭니다. 


그래서 다양한 시제 + 관계대명사를 이용해서 만들어 주어야 합니다. 


영어 문법이 많이 부족하다면 어렵게 느껴지실 겁니다. 


모든 관련 문법을 자세히 설명할 수는 없다는 거 이해해 주세요. 




1. 오랜만에 ~ 했어(다) / 오랜만에 ~할 거야 


I + 과거동사 which (who) hadn't PP for(in) a long time. 

= 오랜만에 ~ 했어.  

(이미 일어난 일이기에 주절에 과거동사를 쓰고 관계대명사 절에는 

그 과거보다 더 과거시점부터 못했던 일이기에 대과거(had pp)를 씁니다.)



I'm going to 동사 (which, who) haven't PP for(in) a long time. 

= 오랜만에 ~할 거야. (할 예정이야) 

(미래의 일이기에 be going to 쓰고 관계대명사 절에는 

현재까지 못한 일이기에 현재완료 have pp 씁니다.) 


ex) 


I played soccer last weekend, which I hadn't done in a long time

/ 난 지난 주말에 오랜만에 축구했어.  


I had Indian food which I hadn't had for a long time. / 나 오랜만에 인도 음식 먹었어. 


I'm going to see my cousin who I haven't seen in a long time. 

/ 난 오랜만에 사촌을 만날 예정이야. 

= I'm going to see my cousin and I haven't seen her in a long time. 


For the first time in a long time, I felt peace in my mind. 

/ 정말 오랜만에 마음의 평화를 느꼈어. 

(for the first time in a long time = 정말 오랜만에 / for the first time 이 '처음으로' 

라는 뜻이기에 정~말 오랜만이라는 것을 강조한 표현) 


He played baseball for the first time in 5 years. 

/ 나 5년 만에 야구했어. 






2. ~한지 오래됐어. ~안 한지 오래됐어.   


It has been (a long time, 시간) + since + (주어 + 과거동사 or have pp 둘 다 가능!) 

/ ~한지(못 한지) ~기간이 되었어. 


= I haven't p.p +  in(for) a long time, in(for) ages, in(for) forever. 

  /~를 오랫동안 못했어. 



ex)


It's been a long time since I had 치킨.  /  치킨 먹은지 진짜 오래됐어. 

= I haven't had 치킨 in a long time.  / 치킨 오랫동안 못 먹었어. 


It's been a while since I met my ex-girl friend. / 전 여친 만난지 꾀 됐다. 

= I haven't met my ex-girl friend in a long time.(for a while) 

/ 전 여친 오랫동안 못 만났어. (못 만나지 좀 됐어)


It's been 3 years since I went there. / 거기 간지 3년 됐어. 

= I haven't been there for 3 years.  / 거기 3년 동안 못갔어. 


It's been 5 years since I haven't smoked. / 나 담배 안 핀지 5년 되었어. 

= I haven't smoked for 5 years. / 나 5년 동안 담배 안폈어. 


It has been a long time since I heard from him. / 그에게서 소식을 들은지 오래됐어. 

= I haven't heard from him for a long time. / 난 그한테 오랫동안 소식을 못 들었어. 



A: Hey! How are you? / 야! 반가워 잘 지냈어? 

B: Wow. I didn't expect to see you here. / 와 여기서 널 만날 줄 몰랐어. 

A: Me neither. I haven't seen you for so long. / 나도야. 너 진짜 오랫동안 못 봤어. 

   How have you been?  / 그 동안 어떻게 지낸거야?  




Tip!) 

How are you? / 잘 지내지? (안녕하시죠?) 


How have you been? / 그 동안 잘 지냈지? (오랜만이네요 안녕하셨죠?) 






3. 오랜만에 ~ 하자 



"~오랫 동안 못했으니까 ~ 하자" 이렇게 표현해주시면 됩니다. 



ex) 


Let's grab some beer. We haven't had it for a long time. / 오랜만에 맥주나 마시러 가자. 


Let's go to the movies. We haven't seen any in a long time. / 오랜만에 영화 보러가자. 







4. In a long time  VS  for a long time 


위에서 이미 ( ) 괄호로  in 과 for 가 같이 쓰일 수 있음을 알려드렸지만 


정확히 언급하자면 두 표현다 뜻은 똑같습니다. 


저는 개인적으로 in 을 더 자주 접했습니다.


ex)


I haven't seen her in ages. (for ages) / 난 그녀를 진짜 오랫동안 못봤어. 








5.  for a long time 만 쓰는 경우! 


긍정문일때는 for!!! 


I've studied English for a long time. / 나 영어공부 오랫동안 해왔어.



wait + for a long time


I have been waiting for a long time. / 나 오랫동안 기다리고 있어.
