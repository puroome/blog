영어로 해내다? 3가지 다른 표현 (make it, made it 뜻 = 만들다가 아니다?) 


 





1. I did it !


하다 = do 입니다. 


했다(하였다) = did 입니다. 




I did it. 이라고 말하면


단순히 '내가 그걸 했어." 라는 표현도 가능하지만 


"내가 해냈어." 라는 의미도 표현가능 합니다. 




ex)

We did it! = 우리가 해냈어!  










2. I made it ! 



이 표현도 단순히 "내가 그걸 만들었어." 라는 기본 뜻이 있습니다. 


하지만 다른 표현으로는 


'해내다, 성공하다, 제때 도착하다(오다) 등..' 의 표현이 가능합니다. 


I did it 보다 더 넓은 의미로 사용할 수 있습니다. 





ex)


Yes! We made it ! / 좋아! 우리가 해냈어! 



Trust me. We can make it. / 날 믿어. 우린 할 수 있어. (해낼 수 있어.) 



I hope I can make it someday. / 언젠가 해낼 수 있기를 바래. 



He was a successful basketball player, 

but I'm not sure if he can make it as a professional golfer. 

/ 그는 성공적인 농구선수 였지만, 그가 프로 골퍼로서 성공할지 (해낼 지) 잘 모르겠다. 



Can you make it? / 올 수 있어요? 

* 그것을 만들 수 있어요? 라는 뜻도 당연히 있습니다. 

  하지만 상대방과 약속이 있는 상황에 이런 대화가 오고간다면

  올 수 있어? / 올 수 있어~ 이런 뜻으로 쓰이는 겁니다. 

  make it 은 쉽지 않게 (힘들게) ~로 가다, 약속장소에 가다(오다)의 느낌이 있습니다.  


  Can you come? 도 동의어 표현이지만 Can you make it? 이라고 하면

  좀더 조심스럽게 물어보는 겁니다. 

  (당신에게) 힘들수도 있지만 올 수 있어? 이런 느낌입니다. 




Can you still make it? / 여전히 올 수 있는거야? 

* 상대방이 갑자기 무슨 일이 생겨서 올 수 있을지 없을지 모르는 상황이라면! 

  


I don't think I can make it. / 나 못 갈 것 같아. 

* 노력은 해봤지만(해보겠지만) 못 갈 것 같다의 느낌을 표현! 



I made it !/ 나 왔어 !

* 나 드디어 도착했어. 

  오는데 좀 힘들었어를 내포하는 표현. 

  쉽지 않게 왔어~ 의 느낌. 



I'm sorry. I can't make it at 5 pm. / 미안해. 5시 까지는 못 갈 것 같아. 



I don't think I can make it to the meeting. / 내 생각에 모임에 제때 못 갈 것 같아.

*make it to 장소 = ~에 (제때) 도착하다 




Can we make it at 2 pm?  / 우리 2시로 할 수 있을까? (변경할 수 있을까?)

* 이 경우에는 (시간) 이 괜찮아? 라고 제안하거나 시간을 바꾸자는 뜻입니다. 

   = Can we change it to 2 pm? 






 





3. pull something off 



= to get something done / winning something difficult 

 (~를 해내거나 / 쉽지 않은 무언가를 이루다, 성취하다) 




I did it! 혹은 I made it ! 처럼 


어떤 승리(?)의 순간에 I pulled it off! (X) 라고 말하면 조금 어색합니다. 


그렇지만 그런 상황을 제외하고는 대체적으로 동등하게 사용할 수 있습니다. 



또한 I did it 과 I made it 은 ~을 해냈는지 이미 내포하고 있지만 


Pull off 는 해낸것이 무엇인지 


pull off something / pull something off 의 형식으로 목적어(~를) 문장에 


포함시킬 수 있습니다. 





ex) 

We can pull this off. / 우리 이거 해낼 수 있어. 



양학선 pulled off his new 1080 skill. / 양학선은 그의 새로운 1080도 기술을 해냈다. 



I never thought you'd pull it off. (출처: 옥스퍼드) 

/ 네가 그것을 해내리라고 생각하지 못했어. 

* think 과거 = thought / will 과거 = would 

  think that 주어 + 동사  구조에서 thought 가 과거로 되어서 

  will 도 시제를 맞춘 would 가 된 구조입니다. 



They nearly managed to get the loan but just failed to pull it off. 

(출처: Macmillan dictionary )

/ 그들은 거의 대출을 받을 뻔 했지만 결국 해내지 못했다. 


* 이 표현에서는 '해내다'의 비슷한 표현이 하나 더 있습니다. 

  manage to 동사원형 = ~를 간신히(가까스로) 하다 

  좀 더 고급스러운 표현으로 사용하기 좋습니다. 

  manage = 경영하다 뜻이 있기 때문에 어떻게 잘 경영해 나가냐, 

  처리하냐 정도로 이해하셔도 됩니다. 

  ex)

  How did you manage to stay slim? / 어떻게 날씨한 몸매를 유지하세요? (출처: 옥스퍼드)

  


If you pull this off, I will pay you double. 

/ 네가 이걸 해내면 내가 2배로 돈 줄게. (보상할게)
