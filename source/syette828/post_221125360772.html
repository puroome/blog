영어로 "쪽수(인원)를 맞추다 / 나 탈 자리있어? (자동차)"  


 



1. 쪽수를 맞추다. 



make the number even / 숫자를 짝수로 맞추다. 


보통 놀이기구를 타던 뭘 하든 짝수로 해야할 경우가 많죠?  


홀수는 'odd' 짝수는 'even' 입니다. 



ex) 


We need you to make the number even. Can you come? 

/ 우리 짝수 만들려면 너가 필요해. 올 수 있어?  



We need one more person to play this game. 

/ 이 게임하려면 한 명이 더 필요해. 





 



2. 자동차에서 한 사람 탈 공간(자리) 


자동차에서 공간이나 자리는 'room' 으로 표현할 수 있습니다. 


ex) 


There's room for one more person. you can join us. 

/ 한 명더 탈 수 있는 자리 있어. 같이 가도 되.  


Do you have enough room back there?  / 뒤에 자리 충분히 있어? 


We only have room for one person, Sorry. /  우리 한 명 밖에 더 태울 수 없어, 미안해. 




오늘은 간단한 표현 배웠습니다.^^
