영어로 '고자질하다, 일러바치다, 고발하다, 위에 보고하다'  







1. 고자질하다 


주로 아이들이 선생님에게 혹은 부모님에게 하는 고자질, 일르다 


tell on + 사람 = ~를 고자질 하다 





(사진출처: 캠브리지사전) 


뜻: 누군가의 나쁜 행동이나 비밀에 관한 정보를 제공하는 것






ex)

If you don't stop hitting, I'm going to tell on you. 

= 너 계속 때리면, 나 너 일러바칠거야. 












~에게 ~를 일러바치다 = tell 사람 on 사람 


on + 뒤에 나오는 사람을 일러바친다는 겁니다. 


ex)


I'm gonna tell teacher on you. = 나 선생님한테 너 일를거야. 

(선생님한테 고자질할거야.) 














2. 고발하다, 회사(윗사람)에 보고하다 


위 표현은 아이나 학생들이, 들에게 사용하기 좋은 표현이고


회사에서 사용하기 좋은 표현은 


write someone up 입니다. 







(사진출처: 캠브리지사전) 


뜻: 규칙, 규율을 지키기 않은 것에 대해서 ~를 보고하는 것 






위에서는 전치사 'on' 을 사용하였고 


이 부분에서는 'up' 을 사용하였네요. 




on 은 ~에 대해서 .. 라는 의미가 있기에 붙여졌다고 외우시고


write someone up 은  윗 사람에게, 위에 보고하기에 


'up' 이 사용되었다고 이해하시면 됩니다. 










ex)

I'm gonna write him up. = 난 그를 고발할거야. 



I wrote him up for sexually harassing me.

 = 난 그가 날 성추행 한 것을 보고했어. 



I was written up. = 나 고발당했어.
