영어로 '그건 안쳐줘, 무효야, ~인 걸로 치지 않아' 





 


오늘은 간단한 회화 표현하나 배워볼게요.


It(That) doesn't count. 입니다. 






이 표현은 뭔가를 비교할 때 어떤 것이 ~의 종류에 속하지 않는다는 것을 


표현할 때 즉, "그건 안쳐줘, 그건 아니지, 그건 속하지 않지, 그건 무효지" 


라는 뜻으로 쓰입니다. 





count 가 동사로 '(숫자를) 세다' 정도로 알고 계시죠? 


count 를 '여기다, 간주하다' 로 해석할 수도 있습니다. 






ex) 

You touched the line so it doesn't count. 

/ 너 금 밟았으니까 그건 무효야. (안 쳐줘) 



If you have no proof, it doesn't count. 

/ 네가 증거가 없으면, 그건 쳐줄 수 없지. (무효지) 



What!? You proposed to me? It doesn't count because I was drunk. 

/ 뭐?! 너 나한테 프로포즈 했다고? 내가 취했었으니까 그건 무효야. 



I only dated her online. Can I still count her as a girl-friend? 

/ 나 그 여자랑 온라인에서 밖에 못 만났는데. 그래도 여자친구로 칠 수 있나? (여길 수 있나?)

* count A as B = A 를 B라고 여기다, 치다 

  A count as B = A를 B라고 여기다 



I don't think they count it as a question. 

/ 내 생각에 그들이 그걸 질문이라고 여기진 않을거야. 



If you are not doing it in a proper way, that doesn't count. 

/ 네가 제대로 된 방법으로 하지 않는다면 그건 안 쳐주지. 

* 예를 들어, 턱걸이, 푸쉬업, 문제풀이 등..을 제대로 안 했을 때 쓸 수 있는 말 



These gifts count as income for tax purposes.  (출처: 옥스포드) 

/ 세금 계산을 위해서 이 선물들도 수입으로 칩니다. (간주합니다) 








The Noah rule: predicting rain doesn't count, building arks does. 

/ 노아의 법칙: 비를 예측하는 것은 쳐주지 않아, 방주를 짓는 것은 쳐주지. 

* 워렌버핏님이 하신 말로 예측만 하는 건 충분치 않고 행동으로 하라는 말





 




Success doesn't count unless you earn it fair and square. 

/ 네가 정정당당하게 성취한게 아니면 성공은 무효다. 

*unless = if not ~하지 않는한
