여러분 set up이라는 표현은 다양한 의미를 갖고 있습니다. 

 

모든 표현들이 다 유용하게 쓰이기 때문에 알아두시면 좋습니다. 

 


 

1. set up (셋팅하다)

 

가장 잘 아시는 표현일거라 생각되네요. 

 

set up the table  = 테이블을 셋팅하다. 

 

set up the booth  = 부스를 셋팅하다. 

 

.

.

.

 

2. 모함!  (함정)

 

He set me up =  그가 날 모함했다. 

 

It was a setup  = 그것은 모함이었다. 

 

쉽게 생각해서 

 

누군가가 다른사람을 속이기 위해 셋팅을 하고 계획을 한거죠. 

 

 

3. set someone up with someone

 

~를 ~에게 소개시켜주다. (주선해주다.) 

 

 

I will set you up with my friend. 

 

= 너한테 내 친구를 소개시켜줄게. 

 

 

 

 

다음에도 유용한 표현 들고 올게요~~
