영어로  꼭 ~ 해, 반드시 ~ 해, 확실히 해두려고 등.. / make sure 사용법 배우기 


 



make sure 관련 표현은 영어회화를 하면 정~말 많이 듣는 표현 중 하나입니다. 


한국말로는 정확히 어떤 뜻인지, 그리고 사용법을 배워볼게요. 


영영사전으로는 


check something so that you can be sure about it 

/ 어떤 것에 관하여 확실히 알거나 할 수 있도록 확인하는 것 




네이버 사전에는 = (반드시 ~하다 / 확실히 ~하다 ) 로 나와 있습니다. 


make sure 은 ~해야한다는 느낌의 (should, have to, got to ) 와 비슷합니다. 


하지만 좀 더 어떤 상황에 대해서 상기시켜주는 느낌, 즉 잊지 말고 ~해 


의 느낌이 듭니다. 


한국말로는 " 잊지말고 꼭 ~ 해" 와 가장 흡사하다고 생각합니다.  





* 문법형태


make sure (that) 주어 + 동사 / ~가 ~ 꼭 하도록 해, ~가 확실히 ~하게 해  


명령문 형태이며 that 은 일반적으로 생략해서 말합니다. 




ex) 

Make sure you close all the doors before you leave. / 나갈 때 반드시 모든 문을 닫고 나가세요. 


Make sure you read all these example sentences. / 여기 예문들을 꼭 다 읽어보세요. 


Make sure you read this before using. / 사용하기 전에 반드시 이걸 읽으세요. 


Just (for) making sure.  / 그냥 확실히 해두려고. (상대방이 제대로 이해했는지 확인하는 차원에서) 


A: Don't forget to do the homework. Okay?  / 숙제 잊지말고 꼭 해. 알겠지?

B: I know. You already told me many times  / 알았어요. 이미 여러 번 말하셨어요. 

A: Just making sure cuz(because) you never do it. / 확실히 해두려는 거야 니가 절대 안하니까 


Do you get what I'm saying? (I'm)Just making sure. 

/ 내가 하는 말 이해하고 있지? 그냥 확실히 해둘려고 


I'm writing a little note for making sure they don't make the same mistake. 

/ 난 그들이 같은 실수를 되풀이 하지 않게 하기 위해 이 작은 메모를 쓰는 거야. 

(for + 동명사  /  make a mistake  실수하다)  



I wanted to make sure you understand what's going on 

but I guess you already know the whole situation. 

/네가 지금 상황이 어떻게 돌아가는지에 대해서 아는지 확인하려 했는데 

이미 전체 상황을 다 알고 있는 것 같네. 



Hey, Are you mad? I just wanna make sure I'm not offending you.  

/ 야, 화났어?(삐졌어?) 혹시나 너 기분나쁘게 하는 건 아닌지 확인하려고.

 

Make sure no one see you. / 주의에 사람 있나 확인해 봐. (아무도 못 보게 잘 확인해)
