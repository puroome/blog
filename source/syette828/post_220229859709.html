영어로 말이 잘 통하다 / 대화가 잘 통하다 / 로 좋은 표현이 있어서 알려드립니다. 




네이버 사전에 보면 


speak the same language 라는 표현이 나오는데... 


잘 쓰이는 표현이 아니고 조금 진부한 표현이라고 생각하시면 됩니다. 




그렇다면 더 좋은 표현으로는?


be on the same wavelength 입니다. 





영영사전 뜻: 


to think in a similar way and to understand each other well



ex)


우리는 말이 잘 통하다. 


= we are on the same wavelength.



우리는 만날때부터 말이 잘 통하는 것처럼 느꼈다.


we seemed to be on the same wavelength from the moment we met. 




wavelength : 파장 


on the same wavelength : 같은 파장을 느끼다 정도로 직역할 수 있겠네요.
