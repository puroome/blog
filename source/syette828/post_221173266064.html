영어로 "~ 어때/ 어땠어? /어떻게 됬어?" 

( what's ~ like? / How is ~? / How do you like ~ 차이 분석) 






오늘은 영어로 "~어때/ 어땠어?/어떻게 되었어?" 를 어떻게 말하는지 배울 겁니다. 


먼저 말씀드리고 싶은 건 


( what's ~ like? / How is ~? / How do you like ~? / How did ~ go? )


이 네 표현 모두 비슷합니다. 모두 ~ 어땠어?(어때?) 로 해석이 가능합니다.


하지만 미세하게 다른 부분을 알려드릴 거고 어떤 상황에서 어떤 표현이 


더 잘 어울릴 지 다양한 예문으로 최선을 다해서 알려드리겠습니다. ^^

 




1. What's something like? /  ~는 어때? (묘사)


   What's it like to 동사?   / ~하는 건 어떤 거야?  / 어때?  (묘사) 



이 질문은 상대방에게 ~가 어떤지 묘사, 설명 등을.. 듣고 싶을 때 씁니다. 



What's your teacher like? / 네 선생님은 어떤 분이셔? / 어때? 


What's your life in Korea like?  / 한국에서의 삶은 어때?? 


What's it like to be a father?  / 아빠가 된다는 건 어떤거야? (상대의 경험 듣고 싶을 때) 


What's it like to live in Korea as a foreigner? / 외국인으로 한국에서 사는건 어떤 느낌이야? 






2. How is (was) ~ ?  / 어때? (어땠어?) 


이 질문은 단순한 감정,기분 물어볼 때 즉, 좋았는지 나빴는지..어떤지... 


궁금할 때 사용할 수 있고 + 상대방의 의견, 경험도 물어볼 수 있는


가장 범위가 넓은 질문입니다. 




예를 들어) 


What's your teacher like? 은 선생님이 어떤 분인지 묻는 질문이었죠? 


How's your teacher? 는 선생님이 어떤 사람인지로 물을 수도 있고


선생님의 상태가 어떤지 궁금할 때도 물어볼 수 있습니다. 


만약 선생님이 사고로 다치셔서 병원에 입원했다 했을 때 


지금은 어떠셔? (괜찮으셔?)  = How's your teacher now? 



ex)


How's life? / 잘 지내니?  (삶은 어떠니?) 


How's everything?  / 잘 지내니? (모든 게 어떠니?)  


How's your family (doing)? / 가족들은 잘 지내니? 


How was the test? / 시험 어땠어? (시험 잘 봤어?) 


A: How was your day? / 오늘 하루 어땠어? 

B: It was (good, okay, alright). It wasn't bad. /  좋았어, 괜찮았어, 나쁘지 않았어. 


How was school today?  / 오늘 학교 어땠어? 








3. How do(did) you like something?  / ~어때? ~마음에 들어?  


How is(was)~ ?  와 거의 흡사하며 좀 더 상대방의


의견을 듣고 싶을 때, "마음에 들어?" 이런 느낌의 질문입니다. 



How do you like your food?  /  음식 어때? 맛있어? 

= How's your food? 


A: I heard you got the new phone. / 너 새 폰 샀다고 들었어. 

   How do you like it?  / 어때? 마음에 들어? 

B: It's Ok. I like it. / 괜찮아~. 맘에 들어. 



A: I tried Korean food for the first time. / 나 처음으로 한국음식 먹어봤어. 

B: So..How did you like it? (How was it?) / 그래서 어땠어? 






How do you like (음식) ? / ~ 어느정도로 익혀드릴까요? 어떻게 드시나요? ....


이 질문은 음식의 굽기정도/ 요리정도에 관련해서도 사용합니다. 




How do you like your toast? / 토스트 어떻게 드세요? 


How do you like your steak (to be cooked)? 

/ 스테이크 굽기는 어떻게 해드릴까요? (어떤 굽기로 드세요?) 


How do you like your egg(s)? / 계란은 어떻게 요리해 드릴까요? 어떻게 드세요? 











4. How did something go?  / ~ 잘 되었어? (~어땠어? / 잘 된거야?)


go 는 '가다' 라는 뜻이죠? 이 가다가 어떤 일이 '진행되다'의 느낌으로 쓰인 겁니다. 


그래서 ~가 어떻게 진행되었니? / 어떻게 되었어? / 어땠어?  


처럼 쓰이는 겁니다. 

 



I heard (that) you had a job interview. / 나 너 면접 봤다고 들었는데

How did it go? / 어땠어? (잘 됬어?) 


How did things go with her? / 그녀와의 일은 잘 된거야? 

(things는 서로 알고 있는 여러가지 일들을 뜻함)
