단(달콤한) = sweet

짠 = salty

짭짤한 = savoury

맹한 맛, 싱거운 = plain / bland  혹은  ex)  it has no taste. 

매운, 칼칼한, 매콤한 = spicy

시큼한, 쉰 = sour  

새콤달콤한 = sweet and sour

쓴, 씁쓰레한 = bitter

떫은 = puckery

바삭한 = crispy

고소한 = nutty / savory       

느끼한, 기름진 =  greasy, oily 

질긴 = chewy

비린 = fishy

스프같은, 걸쭉한 = soupy, thick

~한 맛의 정도 = the lever of (명사)

ex) the level of spiciness / saltiness /sweetness

 

식감 = texture   ex) I like the texture of this food. / texture is tough 식감이 질기다 

씹다 = chew      ex) chew properly or well 꼭꼭 씹어 먹다

아작아작 씹다 (사탕을) = crunch

빨아먹다 = suck ex) suck the candy

삼키다 = swallow

입맛이 없다 = I lost appetite , I don't feel like eating anything.
