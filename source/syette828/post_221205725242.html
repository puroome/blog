33강. one, it 차이 ?  (one 과 it 구분하기) 


 



32강에서는 a 와 the 의 차이를 알려드렸습니다. 


the 에 대해서 이제 완벽히는 아니지만 어느정도 이해하셨다면 


오늘 배울 one 과 it 의 차이는 그리 어렵지 않을 겁니다. ^^







It ??? 



오늘 배울 'It' 은 사물의 단수를 나타내는 '대명사'로 


쉽게 말하면, 앞에서 언급한 것을 지칭합니다. 


보통 '그것' 이라고 해석합니다. 



'It' 의 사용은 엄청나게 광범위합니다. (가주어, it 강조, 등..) 


하지만 오늘 강의를 이해하시기 위해서 it 은 '앞에서 언급한 것' 


정도로만 이해하셔도 괜찮습니다. ^^





ex) 


A: Did you like the food? / 음식 맛있었어? 

B: Yeh, I liked it. / 어, 그거(음식) 맛있었어. 

*앞서 이미 말한 the food를 반복해서 말하는 것을 피하기 위해 대명사 'it' 을 사용



A: How was the game?  /  게임 어땠어?  

B: It was fun. / 재밌었어. 

* the game = it 



A: How was your day?  /  오늘 하루 어땠어? (오늘 잘 보냈어?)

B: It was good. / 좋았어. 







그 앞서 말한 것이 복수일때는 즉, it의 복수는

'they (주격), them(목적격)' 을 써주면 됩니다. 



ex) 

I met some Koreans at the party and they were so funny. 

/ 나 파티에서 한국사람들을 만났는데 진짜 재밌는(웃긴) 사람들이었어. 



A: How were the games?  /  게임(들)은 어땠어?   

B: They were all great. / 다 좋았어. 

* 여러 게임을 치뤘다면 복수로 만들어줘야 겠죠? ㅎ









One ???


one 은 앞선 말한 것이 아닌 어떤 불특정한 사물,사람을 대신할 수 있는 명사입니다.  

(불특정한 명사를 지칭) 


명사이기에 (주어,목적어,보어) 로 사용할 수 있습니다. 


한국말로 '~것' 정도로 해석합니다. 







one 과 it 의 차이점 


1. 형용사 + one  (O)   /   형용사 + it (X) 


쉽게 말해 one 앞에는 꾸며줄 수 있는 단어가 올 수 있습니다. 




A: Which one is your car?  / 어는 것이 네 차야? 

B: The black one. / 검은 거. 

* Which + one =  어느 것,  black (형용사 검은색의) + one 가능!! 



I don't like this color. Can you show me a red one?  

/ 이 색깔 별론데.. 빨간 거 좀 보여줄 수 있어요?  

* 빨간색의 어떤 불특정한 물건을 보여달라고 하기에 one



Can you show me a different one?  / 다른 것 좀 보여줄 수 있어요?  

* 불특정한 어떤 다른 것을 보여달라고 하기에 one



Can you show me another one? / 다른 것 좀 보여줄 수 있어요? 

* 위 문장과 똑같은 뜻이며 another = an + other의 줄임말로 

  불특정한 '다른 ~ 것' 을 표현할 때 쓰입니다. 



I'm gonna sell this car and buy a new one. 

/ 나 이 차 팔고 새거 살거야. 

* 앞서 말한 car에 대한 반복을 피하기 위해 one을 사용!  a new it (안 됨!! X) 

  앞서 말한 자동차와 똑같은 차가 아닌 불특정한 어떤 자동차를 사고 싶기에 one 사용



A: Which one do you like better(more)? / 어느게 더 좋아? 

B: I like the smaller one. / 난 작은 게 더 좋아. 

* 형용사이기에 비교급 (smaller) + one 도 당연히 가능! 



This is the one that I've wanted to buy. / 이게 내가 그 동안 사고 싶었던 거야. 



One of the best things about Korea is fast internet. 

/ 한국의 가장 좋은 점 중 하나는 빠른 인터넷이다. 

* one 이 많이 사용되는 패턴입니다. 

  one of the 복수명사 + is(was)/동사s = ~한 것들 중 하나는 ~ 다. 



You are the only one that I love. / 내가 사랑하는 사람은 오직 너야. 

* one 은 사람도 가능! 



Are you the one who knows the secret?  /  당신이 그 비밀을 알고 있는 사람인가요? 




 





2. <This, That, The, These, Those> + one  (O),  + it (X) 


구체적으로 어떤 것을 가르키며 말할 때도 one과 같이 쓸 수 있습니다. 




ex) 

I like this one. / 난 이게 맘에 들어. 



Can I have that one?  /  저걸로 해도 될까요?( 사도 될까요?, 주문해도 될까요?) 

* this 보다 좀 더 멀리있는 사물 표현할 때 that ! 



I found this key in the classroom. Is this the one (that) you lost? 

/ 나 교실에서 이 열쇠 발견했어. 이게 네가 읽어버린 거야? 



My house is the one right next to the red one. / 내 집은 빨간 거 바로 옆에 있는 집이야. 








it 의 복수가 they/them 이듯이 one 도 복수형태가 가능합니다. 

ones 입니다. 


ex) 

A: Which ones are yours? / 어떤 것들이 니꺼야? 

B: The ones on the floor. / 바닥에 있는 것들. 



The ones you let me borrow weren't good. / 네가 나한테 빌려준 것들은 별로 였어. 

* 목적격 관계대명사 생략된 문장입니다. 

* 목적격 관계대명사 잘 모르시는 분들은 왕기초영어 29강 참조하세요.



I like these ones. / 나 이것들이 좋아. 



They're the ones who stole my car. / 쟤네들이 내 차 훔친 애들이에요. 

* one 은 사람도 가능! 여러명n이기에 ones 사용

   My mom was eating an hamburger and ( it / one) looked so good. (delicious) 


3. 난 이게 더 좋아. / I like this (it, one) better. 


4. Where is your car? 네 차 어딨어? 

   (it / one) is right there. / 바로 저기 있잖아. 


5. 이 상자 너무 작아. 나 좀 더 큰게 필요해. This box is too small. I need a bigger (it / one).

   책상위에 있는 것들. (They are / The ones) on the desk. 


7. 내 친구중 한 명은 축구선수야. (_____) of my friends (_____) a soccer player. 


8. These oranges are clean. Who washed (them / ones)?  이 오렌지들 깨끗하네. 누가 씼었어?
