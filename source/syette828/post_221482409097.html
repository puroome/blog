영어공부! 후라이드치킨, 홈메이드쿠키, 히든싱어의 공통점은?  






 



후라이드치킨 을 영어로 쓰면 


fried chicken 입니다. 


fry = 튀기다 


fried = fry - fried - fried (동사원형 -과거 - 과거분사)


이렇게 3단 변화하는 동사에서 마지막 pp (과거분사) 입니다. 


pp (과거분사) = 형용사  처럼 뒤에 명사를 꾸며줄 수 있습니다. 


해석을 '~된, ~되어진' 으로 하면 됩니다. 





fry chicken = (스스로) 튀기는 치킨 (X)


fried chicken = 튀겨진 치킨 (O)


이해 하셨나요?  




fry 가 동사로 '~를 튀기다'이기 때문에


"I fried chicken. = 난 치킨을 튀겼어." 가 됩니다. 









Homemade 또한 마찬가지로 형용사로 쓰일 수 있습니다. 


made = make 의 pp (과거분사) 입니다. 


즉, homemade = 집에서 만들어진 


homemade + 명사 = 집에서 만든 ~ 


homemade cookie = 홈메이드쿠키 (집에서 만든 쿠키) 







일상생활에서 쓰는 다양한 단어

ex)


Hidden singer (히든싱어) = 숨겨진 가수 

* hide - hid - hidden 






built-in furniture (빌트인가구) 

* build - built - built 

  이미 안에 만들어진, 지어진 가구나 냉장고를 표현할 때 쓰는 형용사 built-in 







응용을 하면 더 많은 단어들을 만들어 낼 수 있습니다. 


ex)

forgotten memory = 잊혀진 기억


broken arrow = 부러진 화살

제가 pp (과거분사) 는 = 형용사처럼 이해하면 된다고 했죠? 


먼저 형용사 문장을 하나 써드릴게요. 


She is beautiful. = 그녀는 아름답다. (beautiful 형용사 = 아름다운)




그럼 pp (과거분사) 를 형용사자리에 넣으면 됩니다. be동사 뒤에 


ex)


It is cooked. = 그것은 요리가 된다. 


It is imported. = 그것은 수입이 된다. 


This room is cleaned everyday. = 이 방은 매일 청소된다. 


My car needs to be repaired. = 내 차는 수리되어야 해.
