영어로 '후진하다, 후진하는거 봐주다'  (feat.오라이의 뜻? 영어로 오라이?) 





 



오늘은 영어로 '후진하다' 관련표현을 알려드리겠습니다. 



후진하다 = back (a, the, your...) car 


back up a car 또한 자주 사용합니다. 



ex)

I need to back up my car. = 나 후진해야 해. 


I'm not good at backing a car. = 나 후진하는 거 잘 못해. 


I have to back up the car a little. = 나 차 조금 후진시켜야 해. 




back (a, the car) car into + something =  차를 후진하다 ~에 부딪히다



I backed my car into the wall. = 후진하다 차를 벽에 박았어. 


Oh, I accidentally backed into a small tree. = 나 실수로 작은 나무를 뒤로 가다 박았어. 


Can you back me up? = 후진하는거 봐줄 수 있어? 


Can you help me back up the car? = 후진하는 거 도와줄 수 있어? (즉, 봐달라는 말) 


Can you check if I'm hitting anything? = 제가 뭐 들이받는지 좀 확인 좀 봐주세요.  











back up 의 또 다른 뜻! 



1. 지지해주다 




I want to back him up. = 난 그를 지지해주고 싶어. 


* 뒤에서 봐주는 행위가 '지지하다, 서포트하다, 응원해주다' 의 

   

  의미로 사용될 수 있음. 







2. back up + 명사 


뜻 = 예비 + 명사 



back-up plan = 대안, 예비책


back up file = 예비 파일, 백업파일  



back up something = ~를 백업하다 


This is how you back up your phone. = 백업은 이렇게 하는 거야. 


I want to back up my cellphone. = 제 핸드폰을 백업하고 싶어요. 







3. ~ 를 뒷받침해주다, ~맞다고 보충해주다 


It can be backed up by science. 

= 그건 과학으로 뒷받침 될 수 있어. 



There are a lot of proofs that I can back this up. 

= 이것을 뒷받침해줄 증거들이 많다. 







오라이 ??? 



'오라이' 는  'all right. alright' 의 잘못된 일본어식 발음입니다. 


후진할 때 '빠꾸' 라고 하는 이유가 'back' 이 잘못된 발음으로 된 것이고 


그리고 '괜찮아' 라는 표현인 all right 이 '오라이'가 된 겁니다. 


영어로 후진할 때 '오라이' 라는 표현을 


원어민들은 여러 가지로 달리 표현할 수 있지만 


You're good. / keep going / you are safe... 등으로 표현할 수 있습니다.
