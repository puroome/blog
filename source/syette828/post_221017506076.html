'work'의 다양한 사용법 ( 영어로 헬스하다 ?) 


 



'work'라는 단어를 오늘은 완전 분석해보겠습니다. 


영어를 공부하다보면 한 단어가 여러가지 뜻이 있어서 어려워 하셨던 경험 많으시죠? 


그 많은 표현들중 실제로 많이 쓰는 표현들만 묶어서 알려드리겠습니다!  



1. work : (동사)  일하다 


I have to work tomorrow.  =  나 내일 일 해야해. 


What days do you work this week?  =  이번주 무슨 요일에 일해? 


I like working with him. =  나는 그와 일하는 걸 좋아해. 




2. work : 효과가 있다, 해결되다, 문제가 해결되다 등... ( work out 으로도 사용가능)  


Did it work?  = 그거 효과 있었어? 


It will work out = 해결될거야 / 효과가 있을거야 


We can work this out! = 우린 이거 해결할 수 있어. 


It doesn't work that way = 그런 식으로는 안돼. 


This medicine worked very well on me. = 이 약이 내 몸에 엄청 잘 들었어. 


How are things working out for you? =  일이 잘 되고 있는 거야?  


What time will it work for you? (what time is good for you?)  =  몇 시가 좋으세요? 




3. (기계, 물건 등..) 작동되다  


This machine doesn't work.  = 이 기계는 작동이 안된다. 


My watch doesn't work. = 내 시계가 안가. 




4. 운동하다, 헬스하다 (work out)   

  보충설명: 흔히 exercise 라는 단어를 '운동하다'로 알고 계시죠?  이 단어도 물론 쓰이지만 

  '헬스하다' 라는 표현이 굳혀진 한국말이듯이 work out도 미국에서 '운동하다'라는 의미로 흔히 쓰이는 표현입니다. 

  exercise 는 뭔가.. 의사들이 환자들에게 'you should exercise more' 운동 더 하세요 ~ 이런 느낌이랄까요... 



I work out everyday. = 나는 매일 운동(헬스)해. 


Do you ever work out? = 너는 혹시 운동하니?  


Working-out is fun. =  운동하는 건 재밌어.  


You look like you work out a lot  = 너 운동 많이 하는 것 처럼 보인다. 




5. work : (명사) 직장, 일 

팁!  work 가 명사로 쓰일 때는 전치사 뒤에 나오는 경우가 많습니다. 



what time do you go to work?  = 몇시에 출근해요? 


I have to get back to work by 8 pm. =  8시까지 다시 일하러 가야해. 


I usually take the bus to work.  =  나는 보통 버스타고 출근해. 


What time do you come from work?  = 몇 시에 회사에서 돌아와? 


I'm getting ready for work. = 일 하러 갈 준비하고 있어. 


Keep up the good work.  =  수고해, 계속 열심히 하세요 (지금 하는 일 계속 잘 하시길 바래요.. 이런 의미입니다.)  


I'm at work.  =  나는 직장이야, 나는 근무 중이야 


I saw her at work. = 직장(회사)에서 그녀를 봤어. 


There is a lot of work to do (I have a lot of work to do) =  할 일이 많아.  

(work 셀 수 없는 명사 취급)   

 




6. work on : ~에 애쓰다, 작업하다, 착수하다 


I'm still working on it.  = 나 아직도 이거 하는 중이야. 


Are you working on your homework? =  숙제하는 중이니? 


We are working on a big project now. =  우리는 지금 큰 프로젝트를 진행 중이야. 




7. 기타 표현 


working hour : 근무시간


hard working person : 열심히 일하는 사람


working environment : 근무 환경
