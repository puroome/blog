영어로 '꼭'?  꼭 그렇진 않아 / 꼭 지켜 / 꼭 이겨줘 / 꼭 만나봐 






 






우선 한글 뜻 먼저 알고 갈게요. 




'꼭' 의 한글사전 뜻:  


어떤 일이 있어도 틀림없이 






우리가 생활을 하면서 이 '꼭' 이라는 부사를 아주 자주 사용합니다. 


예를 들이, "이번엔 꼭 약속 지켜"  /  "꼭 그렇진 않아"  / "이건 꼭 해야해" 등등....



하지만 언제나 마찬가지로 이 '꼭' 이라는 단어도 


여러가지 형태로 다르게 표현됩니다. 



제가 한 단어로 표현된다면 포스팅을 하지도 않겠죠? ㅎㅎ


배우고 나서도 100% 갈증해소가 안될 수도 있지만 


이 포스팅을 보지 않는다면 항상 목마른 상태로 남아있을 겁니다. 






그럼 예문으로 설명드리겠습니다. 




ex)


Make sure you keep the promise this time. 

/ 이번엔 꼭 약속 지켜 


* 꼭 = 반드시  로 바꿔서 표현가능합니다. 

  make sure + that 절  =  반드시 ~ 해라 (잊지말고 ~ 해) 

  또한 promise 는 '새끼손가락 약속(서약,맹세 등..)' 입니다. 

  ~와 만나는 약속으로 쓰이지 않습니다. 




Make sure you do the homework before watching YouTube.

/ 유튜브 보기전에 꼭 숙제해라. 








6 things you must do in Barcelona! / 바르셀로나에서 꼭 해야 할 6가지!



You have to(must) be there on time. Okay? / 꼭 약속시간 지켜. 알겠지?  

* on time = 정시에 (제때) 

* 약속의 중요성 혹은 '꼭'을 얼마나 강조하느냐에 따라서 

   should < have to < must 강도로 사용하시면 됩니다. 




This time you should really see him. He is really good looking.

/ 이번엔 너 진짜 걔 꼭 만나봐야 해. 진짜 잘 생겼어. 




It's not always like this. / 그게 항상(꼭) 이렇지는 않아요. 







You don't necessarily have to do this now. / 너 이거 지금 꼭 해야 하는 건 아냐. 


* necessarily = 반드시 라는 부사로 

  일반적으로 이렇게 not 과 같이 쓰입니다. 

  꼭 ~할 필요는 없어,  꼭 ~이지는 않아 등... 의 형태에 아주 잘 맞는 부사입니다! 

  단답형으로 "Not necessarily." = "꼭 그렇진 않아"  이렇게 말하기도 합니다. 




It's been proven. older doesn't necessarily wiser. 

Some people never learn.

/ 지금껏 증명되었다. 나이가 더 많다고 꼭 더 현명하지는 않다. 

몇몇은 평생 배우지 않는다(발전이 없다)




I'm sure he will make it this time. / 그는 이번엔 꼭 해낼거야. 


* 이렇게 미래에 "꼭 ~ 일거야" 의 형태는 

  위에서의 상황과는 또 다릅니다. 

  necessarily, make sure, have to .. 모두 안 어울립니다. 



No matter what happens, I will make it happen. 

/ 무슨 일이 있어도, 꼭 해낼 게. 








꼭 이겨보고 싶다. / I really want to win. (I want to win so bad.) 



꼭 이겨줘. / Please win (the game). 

 


꼭 좀 해줘. / Please do it (for me).

* 이런 식으로 부탁할 때 '꼭' = 'please' 입니다.
