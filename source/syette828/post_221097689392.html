영어로 "끝났어, 끝내다, 끝났어?" (2탄) I'm done 과 I have done 차이? 


 

(사진출처: 구글) 



앞서 1탄에서 기본적인 '끝내다' 관련 표현을 배워봤습니다. 


1탄에 이어서 2탄 이어갈게요~ ^^ 





4. I am done 과 I have done 은 같을까?? 

 


I am done 은 단독으로 쓰여서 "나 끝냈어" 라고 할 수 있지만 

 

I have done 은   끝내다 라는 뜻보다는 " ~를 했다" 로 해석합니다.

 

또한 단독으로 "I have done" 이렇게 쓰이지 못하며 뒤에 목적어가 들어갑니다. 

 


ex)  

 

I have done my homework.  / 나 숙제했어.  ( 끝마쳤다는 느낌보다는 했어 의 의미가 강함)  


I haven't done anything yet. / 나 아직 아무것도 못했어. 


I've done something for her.  /  난 그녀를 위해 무언가를 했어. 

 

 


네이버 사전이나 다른 글에서 

 

I have done with you.  / 너랑 끝났어.  (x) 


라고 나오는데 이 표현은 잘못된 표현입니다! 

 

I'm done with you. (올바른 표현)

 

 


또한  have + p.p는 경험을 뜻하기도 하기 때문에 


"~ 해봤어" 라고 해석할 수 도 있습니다. (상황에 따라) 



ex)


I've done this before.  /  나 전에 이거 해봤어. 


I've done it many times. /  나 이거 많이 해봤어. 


Have you done this ?  /  너 이거 해봤니? 





5. Something is done.(finished)   / ~가 끝나다 


 

사람만 뿐만 아니라 앞에 사물도 주어가 될 수 있겠죠? 


 

I'll call you when the work is done.  /  일이 끝나면 연락드리죠. 

 

My homework is done.  / 내 숙제 끝났어. 

 

This project will be finished(done) by next month. /  이 프로젝트는 다음달까지 끝날거야. 

 

 

 



6. Something is done. 


(수동태: ~가 되다, 행해지다, 이루어지다,  조취가 취해지다, 등...) 

 

 


Everything is done by me here. /  여기서 모든 일은 나로 인해 이루어진다. 

 

The purchasing process is done on the internet. /  구매과정은 인터넷에서 이루어 집니다. 

 

Something has to be done. /  무슨 일이라도 행해져야 한다. (have to ~해야한다) 

 

It shouldn't be done this way.  /  이런 방식으로 일이 진행되면 안됩니다.
