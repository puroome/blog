영어로 '더 이상 ~ 아니다' (no longer / not anymore 차이와 사용법 배우기)  





 



no longer = not anymore 



차이는 no longer 이 좀 더 formal 합니다. (not anymore 이 좀 더 자연스럽게 들림) 


또한 not anymore 은 단답형으로 사용가능합니다! 




ex) 


I'm no longer a student. / 난 더 이상 학생이 아니다. 

= I'm not a student anymore. 



She doesn't like me anymore. / 그녀는 더이상 나를 좋아하지 않는다. 

= she no longer likes me. 



I no longer have that problem. / 나 더이상 그 문제 없어. 

= I don't have that problem anymore. 



We are no longer friends. / 우리는 더이상 친구가 아니야. 

= We are not friends anymore. 








When we are no longer able to change a situation - we are challenged to change ourselves. 

/ 우리가 더이상 주변 환경,상황을 바꿀 수 없을 때 - 우리는 우리 스스로를 바꾸도록 요구되어 진다.




It isn't fun anymore. / 그건 더이상 재미있지 않아. 

= It's no longer fun. 



A: Do you like me?   /  너 나 좋아해? 

B: Not anymore   / 아니 더이상 좋아하지 않아. (이젠 좋아하지 않아.)
