영어로 "끝났어, 끝내다, 끝났어?" (1탄) done? or finished? 


 

(사진출처: google)




오늘은 실용성 끝판왕!  '끝내다' 관련 표현 배우겠습니다. 




1. I'm done. 


I'm done 은 "나 끝났어 / 나 끝냈어 / 전 끝났어요" 의 의미로 쓰입니다.  



(be + p.p (수동태) 형식으로 직역하면 뜻이 이상해집니다. 


그래서 관용적 표현이라 생각하시고 외워서 사용하시는 게 좋습니다.)   



I'm done 이 단독으로 쓰이는 경우를 보여드릴게요. 




이거 사용 다 하셨어요?  → Yes, I am done. 



오늘 일 끝났어?  →  Yeah, I'm done. 



숙제 다 했니? /끝냈니?  →  Yes, I'm done. 



반대로 질문도 단독으로 쓰입니다. 


Are you done?     다 쓰셨어요? / 다 끝났어요? / 다 하셨어요? 


Are you done?     주문다 하셨어요? (커피숍에서 앞사람이 주문 다하고 서있을 때) 


Are you done?     기구 다 쓰셨어요? (헬스장에서 이 사람이 기구를 다 썼는지 물어볼 때)


Are you done?     .... 기타 모든 상황 응용 가능!  




2. I'm done  +  ??  


I'm done + 동사 ing.  =  ~ 하는 거 끝났어. 


Are you done + 동사 ing =  ~하는 거 끝났어?  


Tip)  with와 함께 쓰이기도 합니다. (I'm done with + 동사 ing)




ex) 


I'm done working. / 나 일 끝났어. 


I'm done with studying today. /  오늘 공부는 끝!  


I'm finally done cleaning the house. / 이제서야 (마침내) 청소 다 끝났다! 


I'm already done eating.  /   나 이미 식사(먹고 있던 것) 했어. (다 먹었어.) 


I'm done arguing with this. / 이걸로 다투는거 이제 끝이야. (argue 논쟁핟, 말다툼하다) 




Are you done talking?  / 이제 할 말 다 하셨어요? (비꼬는 말투로) 


Are you already done packing? /  이미 짐 다 쌌어? 


Are you done reading the book I let you borrow?  /  내가 빌려준 거 다 읽었어? 


If you are done using it, Can I use it now?  /  다 사용하셨으면 제가 사용해도 되죠?  




I'm done with + 명사     


Are you done with + 명사 




I'm done with you.  /  너랑은 끝났어. (더 이상 대화하고 싶지 않을 때/ 헤어지자는 의미로 가능) 


Can you put it back if you are done with it?  /  그거 다 썼으면 제자리에 놔줄래? 


I'm done with the dishes. = I'm done doing the dishes. /  설거지 끝났어. 


She's almost done with her makeup /  그 여자애 화장 거의다 끝났어. 


Are you done with that coffee?  /  저 커피 다 마셨어요?  


Are you done with homework? / 숙제 다 끝났니?  


Are you done with her? / 그 여자랑은 끝난거야? 


Are you done with your lunch? / 너 점심 다 먹었어? 


 

(사진출처: google)



3. I am done = I am finished 


두 표현은 동의어로 아무거나 사용하셔도 됩니다. 


하지만 일반적으로 I'm done 을 더 자주 들으실 겁니다. 


I'm finished 역시 똑같은 패턴으로 문장을 이어주면 됩니다.
