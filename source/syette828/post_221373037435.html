영어로 알람을 ~시에 맞추다, 알람을 끄다, snooze button? 






우리가 알게 모르게 사용하는 영단어가 많습니다. 


사실 '알람' 또한 영어입니다. 


'경보, 경보신호' 의 뜻을 가지고 있습니다. 




오늘은 알람시계 관련하여 알람을 설정하다, 맞추다, 


끄다, 그리고 snooze button 에 대해서 배워보도록 할게요. ^^



 





1. 알람을 ~시에 맟추다 



set the(an) alarm for 시간 


왜 for + 시간 일까? 


보통 시간 앞에 at 이라는 전치사를 붙입니다.


set the alarm at 6 라고 한다면  


내가 6시에 알람을 맞춘다 라는 뜻이 될 수 있기에 


6시에 일어나기 위해 (for) 를 써주는 것입니다. 


제가 생각한 논리이지만 이해하기 어려우시면 그냥 외우셔도 되요~ 

 



ex)

Can you set the alarm for 6? / 6시에 알람 좀 맞춰줄 수 있어? 


I set the alarm for 7 o'clock. / 나 7시에 알람 맞췄어. 

* set 의 과거는 set 


What time did you set an alarm for? / 몇시로 알람 맞췄어??


You mean you want to set this alarm for pm or am? 

/네 말은 이 알람 오후에 맞추겠다는 거야 아님 오전에 맞추겠다는 거야? 









2. 알람이 울리다 


(the) alarm goes off 



ex)

Alarm went off at 6. / 알람이 6시에 울렸다. 


Alarm didn't go off. / 알람이 울리지 않았어.  











3. 알람을 끄다 


turn off the alarm 


알람을 켜다 turn on 이라는 말은 쓰지 않습니다. 


하지만 알람을 끌때는 turn off 라고 할 수 있습니다. 



ex)

Can you please turn off the alarm? 

/ 제발 알람 좀 껴줄 수 없니? 








4. 스누즈 버튼 (snooze button) ??




snooze button 은 한국말로는 없습니다. 


한국말로 없는 단어들이 종종 있는데 


신기하게도 이 단어가 그 중 하나입니다. 



빵 봉지 묶을 때 쓰는 것이라던지 


피자에 올라가는 삼각대 모양의 이름모를 것~ 


요런 잡다한 건 이름없이 쓸 수도 있지만 


snooze button 은 뭐라해야 하는지.. ㅋㅋㅋ




snooze button 이란? 


알람을 설정하고 알람이 울릴 때 이 snooze 버튼을 


누르면 잠시 후에 (보통 5분) 다시 울립니다. 


좀 더 자고 싶을 때 누르는 버튼이라고 생각하시면 되겠네요. 



제 핸드폰도 이 설정이 되어 있어서 5분 간격으로 자꾸 울릴 때가 있어요. 


제가 스누즈 기능을 은근 즐기는 것 같기도 해요. ㅎㅎㅎ



ex)

How many times did you hit the snooze button? 

/ 몇 번이나 스누즈 버튼 눌렀어??




<Amazing snooze proof alarm (놀라운 스누즈 방지 알람)>


스누즈 버튼을 자꾸 눌러서 늦게 일어나는 것을 방지하기 위해 


발을 딛고 올라서야 알람이 꺼지는 기계입니다.
