영어로 "빨리 다녀와 / 빨리 갔다와 / 최대한 빨리 와 등.."  뭘까? (feat. 빨리해) 


 



며칠 전에 댓글로 들어온 질문인데 많은 분들이 같이 아셨음 해서 포스팅합니다. 



빨리 다녀와, 빨리 갔다와 영어로 바꾼다고 무조건 


fast 와 quick 을 사용하는 건 아닙니다. 






1. Hurry (Hurry up) ﻿


일반적으로 '서둘러' 로 사용하지만 상황에 따라 '빨리다녀와' 로도 해석됩니다. 


Hurry 와 Hurry up 둘다 뜻은 똑같습니다. 



ex) 


You should hurry. / 서두르는게 좋겠어. (빨리 갔다와야 해) 

= You gotta hurry. (gotta = have to / should 보다는 좀 더 '빨리 오라는' 의무성을 강조) 



A: Can I go to the restroom?  / 화장실 다녀와도 되요? 

B: Yes, but you should hurry (up). We don't have time. / 어, 근데 빨리 다녀와. 우리 시간 없어. 






2. fast 와 quick 이용하기 


한국말로 '빨리 다녀와, 갔다와' 는 가는 움직임을 빨리 하는 것에 집중하였다면


영어는 가서 '무언가를 하는 행위' 자체에 더 집중합니다. 


그래서 go(가다), get back, come back (돌아오다) 라는 단어로 표현하지 않습니다. 




ex) 


Get it done quickly. / 빨리 끝내. (즉, 빨리 일 처리하고 돌아와.) 


Make it quick. (fast) / 빨리 해. 

(make 는 만들다 외에 '~을 해내다, 시간맞춰 오다' 의 뜻이 있습니다. ) 


Do it fast. (quickly) / 빨리 해. 




'빨리해' 라는 표현은 이미 포스팅 한 적이 있지만 


do it quickly, make it quick, hurry up  모두 가능하며 


get it done, get it done. 을 빨리 말해도  '빨리 빨리' 라는 느낌으로 의미가 전달 됩니다. ^^







3.  as soon as 주어 + 동사  


as soon as 주어 + 동사  = 주어가 동사 하자마자  



주의할 점은 이 표현은 "빨리 다녀와" 라는 뜻으로 잘 쓰이지 않습니다. 


다만 다른식으로 응용이 많이 되기에 관련 표현으로 알려드립니다. 



돌아오다는 'come back' 보다는 'get back'을 원어민들은 더 많이 사용합니다. 



ex) 


Get back (here) as soon as you are done. /  끝나자마자 돌아와. 


Get back home as soon as school's over. (school finishes). / 학교 끝나자마자 집에 돌아와라. 


Get back as quickly as possible. / 가능한 빨리 돌아와.
