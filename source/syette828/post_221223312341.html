Koreanized, Americanized /한국화, 미국화? 미국화시키다?

(그 외에 나라들은 어떻게?) 



 






Korean, American 등...+  ized 를 붙이면  ~화된 (수동태)  이란 뜻으로 쓰입니다. 



Americanized = 미국화된 (발음 : 어메리카나이즈드) 


Koreanized = 한국화된 (발음: 코리아나이즈드) 




+ ize 를 붙이면 동사형태로 Americanize = 미국화시키다 



+ nization 을 붙이면 Americanization = 미국화 (발음: 어메리카나이제이션) 명사가 됩니다.  








 




ex) 


This is Americanized 비빔밥. = 이건 미국화된 비빔밥이야. 

= This is American version of 비빔밥.  

= This is Americanized version of 비빔밥.

= This is American style 비빔밥.  



They're trying to Americanize me. / 그들은 나를 미국화하려고 한다. (미국사람처럼 만들다.) 



He got Americanized. / 그는 미국화되었어. 



He acts and talks like American. / 그는 미국사람처럼 행동하고 말한다. 



You're becoming(getting) Koreanized. / 너 점점 한국화되어가는 것 같아. 







americanized, americanization 등 ...은 사전에 등록이 되었지만 


다른 나라들은 그냥 만들어 쓰는 경향이 있습니다. 


그래서 모든 나라가 가능한 것도 아니며 n 으로 끝나지 않으면 


발음이 이상해서 잘 쓰지 않는 것 같습니다. 







Mexican → Mexicanized (가능) 


Japanese → Japan → Japanized (가능)  


Chinese → china → oriental or Chinese version of ~ (불가능하기에 다른형태로)
