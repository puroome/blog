사기도 쉽게 치는 것은 아니다. 틀을 잡고 계획을 세우고 해야 한다. 그런 의미에서 누군가에게 짜고 치는 사기에 당했다면 You’ve set me up.이라고 말한다. 영화에서 그런 장면을 많이 볼 수 있다.

set up의 의미를 살펴보자.

■ set someone/something up

1. ...을 세우다, 정하다, 설립하다

(= erect, establish)

2. 속이다, 함정에 빠뜨리다

(= deceive, frame)

텐트를 세우는 것처럼 사람에게 하나하나 뼈대를 세우고 천을 친다면 그것은 사기를 치는 것이다. 미리 함정을 파고 계획을 세우는 것이다. 그래서 set up이 사람 목적어를 가지면 '속이다, 사기치다'는 뜻이 된다.

​

It took only an minute to set up the tent.

그 텐트를 설치하는 데는 일 분 밖에 안 걸린다.

I was just standing there. Somebody set me up for the robbery.

나는 그냥 거기 서 있었다. 누군가 나를 강도의 함정에 빠뜨렸다.

성공하려면 실패를 해보아야 한다. 실패로부터 배우는 것이 성공의 지름길이다. 위험을 무릅쓰지 않으면 길이 보이지 않다.

그래서 다음과 같은 말이 있다.

The best way to set someone up for failure is to hold them back from taking risks.

- Arianna Huffington

누군가를 실패할 수밖에 없도록 계획을 짜는 가장 좋은 방법은 그가 위험을 무릅쓰지 않도록 붙잡아 두는 것이다. - 아리아나 후핑톤
