■ beef ... up 

   : 강화시키다 

   : strengthen by addition, reinforcement etc. 

﻿back ... up : ...을 뒷받침하다, 지지하다(= support) 

brace ... up : ...을 분발시키다, 힘을 내게 하다(= prop up, support)

shore ... up : (지주를 대서) 받치다, 넘어지지 않도록 강화하다

prop ... up :  (주로 좋지 않은 것을 ) 받쳐서 넘어지지 않도록 하다, 지원하다 

﻿ 

 beef up은 원래 '소고기를 더﻿ 많이 주다'에서 유래한 표현이지만 '자금이나 정책을 더 강화시키다'는 의미로 많이 사용한다. up을 사용한 유사한 표현으로 back up은 기본적으로 '지원하다'이고, brace up은 '용기, 기운 등을 분발하게 하다', shore up과 prop up은 '쓰러지지 않도록 버팀목(shore, prop)을 대서 튼튼히 강화시키기다'는 의미로 활용된다.
