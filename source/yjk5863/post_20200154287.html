■ if any

    설사 있다고 해다 ; 만약 ... (있다면)

    there may be something like that

 

★ if any의 부정 표현이 if not이며, if anything은 문장간에 접속한다.

○There are few if any such books.

설사 있다하더라도 거의 그런 책은 없다.

 

■ if anything

   어느 편인가 하면, 오히려, 아무튼

   You use ‘if anything,’ especially a negative

   statement, to say that something different

   from what you have just said is actually the

   case.

★ 뭔가 부정적인 말을 하기 위해 사용하는 표현으로 방금 말한 것과 다른 어떤 점이 있다는 것을 시사한다.

○He didn't look nervous. If anything, he looked slightly perplexed.

그는 불안해 보이지 않았다. 어떤 편이였나면  그는 약간 당황해 보였다.

 

■ if not

   비록~은 아닐지라도, 만약 ~이 아니라면

   to say that the real state of affairs might ﻿

 

   be different from the one you have mentioned;

 

    perhaps)

★ 실제 말했던 상황과는 뭔가 '아마' 다르다는 것을 말하기 위해 사용하는 표현이다.

 

○The first project can represent hundreds of

thousands if not millions of pounds investment.

첫번기획은 아마 수백만 파운드는 아닐지라도 수십만 파운드의 투자를 나타낼 수는 있다.
