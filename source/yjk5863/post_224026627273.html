1. 기본 의미

count = 세다, 계산하다

2. 의미 확장의 단계적 발전

① 물리적 계산 → 추상적 계산

- count money (돈을 세다)

- count the days (날짜를 세다)

- count your blessings (축복을 헤아리다) - 추상적 개념도 '세어서/헤아려서 아는 것임'

② 계산 가능 → 확실함

- "계산할 수 있다" → "확실히 존재한다"

- You can count on bad weather in winter. (겨울엔 날씨가 나쁠 것을 확신할 수 있다.)

- 예측 가능하고 확실한 것들을 "계산"에 넣을 수 있다는 개념

③ 계산 가능 → 신뢰

- Don't count your chickens before they hatch.

(부화하기 전 닭을 세지 마라.)

→ 불확실한 것은 계산(신뢰)에 넣지 말라는 의미

- 반대로, count on은 "...에 대해 확실히 계산에 넣을 수 있다" → "신뢰할 수 있다"

3. 전치사 on의 역할

on = "~위에, ~에 기반하여"

​

- rely on = 의존하다 (필요에 의한 의존)

- depend on = 달려있다 (조건적 관계)

- lean on = (기대듯) 의존하다

​

이런 표현과 관련하여 count on을 이해하는 것도 가능하다.

​

- count on: "~을 기반으로 계산하다" → "~을 믿고 계획을 세우다"

4. count 관련 다른 표현

​

- count someone/something in = ...을 셈을 할 때 넣다 → 포함시키다 (참여)

- count someone/something out = ...을 셈을 할 때 빼다 → 제외시키다 (배제)

- count for much/nothing= ...인 것으로 세다 → 많은 가치가 있다/전혀 가치가 없다(중요성)

- count against someone/something = ~에 불리하게 세다 → 불리하게 작용하다

5. 현대적 사용과 뉘앙스

 상황별 사용:

- 비즈니스: "We're counting on this deal" (이 거래에 기대를 걸고 있다)

- 일상: "I'm counting on you to be there" (네가 거기 있을 거라 믿는다)

- 부정문: "Don't count on it" (그것에 기대하지 마라)

 감정적 무게:

- count on은 rely on보다 더 개인적이고 친밀한 느낌

- 계산처럼 "확실한" 신뢰를 표현하면서도 따뜻함을 담고 있음

"Love is counting on someone even when you can't count the reasons why."

"사랑은 누군가를 믿는 것이다, 이유를 헤아릴 수 없을 때조차도."

- 사랑은 무조건적 신뢰를 본질로 한다는 말
