■ work vt.

​

work one's face : 얼굴을 씰룩이다/움직이다

work a machine : 기계를 작동시키다

work a miracle : 기적을 낳다 (: produce a miracle by working hard)

= work wonders ​​

work his wife hard : 부인을 혹사시키다

work out a difficult problem : 어려운 문제를 풀다

a jewel wonderfully wrought : 훌륭하게 세공된 보석

work miracles along the Han : 한강을 따라 기적을 만들어내다

work off a debt : 일을 해서 빚을 다 갚다

work one's way up : 열심히 일을 해서 승진하다

work one's way through college : 열심히 일을 해서 대학을 마치다

work ... up : ...을 흥분시키다, 부추기다 ; 자세히 조사하다

​

 동사 work는 자동사로 '주어가 일을 하다', 타동사로는 '목적어를 일을 시키다'는 뜻이다. 위의 표현에서 목적어의 종류에 따라 그 뜻이 변하는 것을 상상하면서 의미를 이해하면 많은 뜻을 쉽게 이해할 수 있다. 예를 들어 '얼굴을 일(work)시키는 것'는 '얼굴을 씰룩이고 움직이는 것'이고 '기계를 일(work)시키는 것'은 '기계를 작동시키는 것'이다. work ... out은 '어떤 것을 일을 시켜 해결 하는 것'이고 work ... off '일을 해서 ...을 다 없애는 것'이다.
