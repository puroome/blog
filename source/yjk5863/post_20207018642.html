■ pull off

   vi. 떠나다, 가다

■ pull ... off

   vt. 완수하다, 성공하다, (옷 등을 급히) 벗다

   = carry ... off, bring ... off

​

pull over/ up : vi. 서다 vt. 세우다

pull away : 떠나다

pull out : vi. 철수하다, 떠나다 vt. 철수시키다

pull through : vi. (병에서) 회복되다

​

 ​동사 pull은 자동사와 타동사 둘 다 가능하다. pull off의 자동사 의미는 '떠나다, 가다'이고, 타동사 의미는 '...을 완수하다, 성공하다, (옷을) 벗다'이다. '계획 등을 완수하다'는 의미로 carry ... off와 bring ... off도 off를 사용한다는 사실을 알면 연관지어 함께 외울 수 있다. 반면 pull over나 pull up은 '마차나 차가 서다' 혹은 '멈추게 하다'는 의미로 자동사 혹은 타동사로 쓰인다. over와 up(위로)를 써서 세우다는 의미가 된 것은 '마차를 세울 때 말의 고삐를 위로 당기기 때문'이라고 한다.

 pull away도 자동사로 '떠나다'는 의미이다. pull out은 자동사로 '군대 등이 철수하다', 타동사로는 '군대 등을 철수 시키다', pull through는 '주어가 통과하다'는 뜻에서 '회복하다'는 뜻이 된다.
