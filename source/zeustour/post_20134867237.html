Go with the flow

 

 

 

go with the flow, 

 

go with the tide,

﻿

join the crowd.

 

 

순리를 따르다,

 

대세를 따르다,

 

시대에 순응하다,

 

시류를 타다 등등...

 

자연스러운 흐름에 맡기다 는 뜻의 영어표현.

 

 

- I just want to go with the flow.

   난 그냥 순리대로 살고 싶어 / 물 흘러가는대로 살거야.

 

- You're no better than anybody else here. Join the crowd.

   여기 너보다 못한 사람이 어딘니. 걍 남 하는대로 해.

 

- I'm willing to go with the flow, but I'm not sure in which direction it is flowing.

   난 순리대로 살고 싶은데 순리가 어느 방향인지 모르겠어.
