I lost my appetite 


우리가 잘 알고 있는 ﻿


입맛을 잃다, 입맛이 없다


식욕을 잃다, 식욕이 없다


라는 뜻의 영어 표현 이죠~


I think I've lost my appetite.

나 식욕을 잃어버린 것 같아


I don't have an appetite.

입맛이 없어요





반대로 입맛이 돌아오다는,


I've got my appetite back.

저 입맛이 돌아 왔어요





그러면 입맛이나 식욕을 돋우다 


는 영어로 어떻게 표현할까요?


We're working up an appetite with fresh spring herbs.

저희는 신선한 봄나물로 입맛을 돋우고 있습니다


work up 은 


무언가를 돋우다, 넘치게 하다 란 뜻이고, 


appetite 는 식욕, 입맛 이죠^^


work up an appetite 가 되면,


입맛을 돋우다,


식욕이 넘치게 하다


라는 의미가 됩니다.



 



You must have really worked up your appetite.

식욕이 정말 넘치 셨었나봐요



work up 대신에 build up 을 쓰기도 합니다


Watching all this fresh air can really build up an appetite.

이렇게 신선한 공기와 자연을 보는 것은 식욕을 넘치게 하지요



 



여기서 잠깐 잘못 쓰고 있는 우리말을 체크하고 가실께요~
