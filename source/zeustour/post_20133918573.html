﻿미드 <Friends> 에서 출산휴가 중에 자기 자리를 떡 하니 차지한 남자에게 따지러 간 레이첼이

 

그와 한참 실랑이를 벌이고 있는데 로스가 조심스럽게 다가와 code brown  상황이

 

발생했다고 말합니다. 뭔가 심상치 않은 일이 벌어진 것 같긴 한데, 대체 무슨 일일까요?

 

 

code brown 은 병원에서 쓰는 code blue 를 패러디한 말이다.

 

병원에서 code blue 라고 하면 환자에게 위급한 상황이 닥쳤다는 뜻이다.

 

심장이 멎었거나 해서 즉각적으로 처치하지 않으면 안 되는 긴급한 상황일 때,

 

code blue 라고 외친다.

 

 

로스가 말한 code brown 은 brown 이 갈색(똥색) 이라는 걸 생각하면 쉽게 그 뜻을

 

짐작할 수 있다. 똥 싼 기저귀, 즉 아이가 기저귀에 똥을 싼 상황이 바로

 

code brown 인 것이다.

 

 

Ross : Rachel, we have a code brown situation.

 

Rachel : Could you please take care of it for me?

 

Ross : All right, but you're gona have to do one sometime.

 

 

로스 : 레이첼, 여기 코드 브라운 상황이 발생했어.

 

레이첼 : 니가 좀 처리해 줄래?

 

로스 : 알았어, 근데 너도 언젠가 한 번은 해야 할 거야.
