this.that,these,those 개념을 알아보자





가까운것 1개(단수)를 가르킬때 this

가까운것1개이상(복수)를 가르킬때 these



멀리 있는 1개(단수)를 가르킬때 that

멀리 있는 1개(복수)이상을 가르킬때 those







[가까운 사물을 가르킬때]



this is + 단수명사.(이것은 ~이다.)

ex) this is hat.


those are + 복수명사. (이것들은 ~이다.)

ex) those are hats.




[멀리 있는 사물을 가르킬때]


that is 단수명사. (저것은 ~이다.)

ex) that is hat.


those are + 복수명사. (저것들은 ~ 이다.)

ex) those are hats.




다르게 사용 가능한 표현법


this + 단수명사 is~.

ex) this hat is good.


those + 복수명사 are~.

ex) those hats are good.


that + 단수명사 is ~.

ex) that hat is good.
