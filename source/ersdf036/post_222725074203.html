what if는 상황에 따라 여러가지 다른 의미를 가집니다.


1.~ 라면 어찌될 것인가


2.~ 하다 하더라도 어쨌단 말인가


3.~ 한들 상관없지 않은가


위 상황의 경우는 3번에 해당한다고 보여 집니다.화자가 그녀에게 말좀 붙여 보고 싶은데 라고 하니까 

 해봐.실패한들 대수야 라고 화자의 행동을 격려,조장하는 말입니다.

what if는 what if you should fail이 줄어든 것으로 봅니다.


if  에 what 을 붙여주면 그게 what if 입니다. ^^


if 는 만약 ~라면 이라는 개념이고


what 은 무엇이냐? 어떻것이냐? 라는 개념이므로



what / if 


어떻게 되는것인가 / 만약 ~라면


위와같은 구조일뿐입니다. 따라서 



what if '만약~라면 어떨까? 만약 ~라면 어쩌지?' 라는 뜻으로 의역이 될뿐입니다.
