There is no,There is not,셀수있는명사,셀수없는 명사일때는?






질문:


there is- 단수명사, there are- 복수명사 이렇게 쓰이는건 알겠는데

셀수없는 명사일때는 어떻게 사용될까요?






답변:


any는 셀 수 있는 명사(복수), 셀 수 없는 명사(단수) 다 쓸 수 있습니다. 


There aren't any books.

There isn't any water.


셀 수 없는 명사는 복수형이 없기 때문에

단수형만 존재합니다.

(셀 수 없는데 복수형이 존재할 리 없겠지요?)



불가산명사를 표현할때 is를 쓰시라는 이유는 셀수없기 떄문에 단수 취급을 하기 때문입니다

둘 다 맞는 표현입니다.



There aren't any books. (o)

There isn't any water. (o)


There isn't a book. (o)

There aren't books. (o)



no를 붙여서 부정문을 만들 수도 있습니다.


There is no book. (o)

There is no water. (o)



요약:


* There is(isn't) + 단수명사 (셀수있는명사, 셀수없는명사둘다가능)

* There are(aren't) + 복수명사(셀수있는명사만 가능)







부정문은 다양한 방식으로 표현할 수 있습니다.


 

There is no와 There is not 다 가능합니다

차이점이라고는 no앞에는 명사가 not앞에는 형용사가 옵니다
