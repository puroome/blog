1) used to + 동사 원형 ; ~~(과거의 습관,상태) 하곤 했다

현재와 상반되는 과거의 습관이나 상태를 나타냄


(예)

My mom used to go to church on Sunday. (지금은 교회 안나간다는 의미가 포함되어 있음)

My mom used to be a baseball player. (지금은야구선수 였지만 지금은 아니라는 의미가 포함) 


 

2) be used to 동사원형 : ~ 하는데 사용되다. 


(예)

The knife is used to slice off fruit.  

칼은 과일을 자르는데 사용된다.


*(수동태) 그냥 특별한게 아니라 이 문장 형태가 수동태임

이 때의  to ～ 는 to부정사로 부사적용법이다.

 


3) be used to + ~ing(동명사) ; ~하는데 익숙해져 있다.

= be accustomed to + ~ing(동명사)

be used to 뒤에는 ~ing가 올 수도 있습니다.


(예)

I'm used to living with many people.

여러사람과 사는것이 익숙해 있다.


I’m used to eating sweet food.

달달한 음식에 익숙하다　



4). get used to 명사： 에 익숙하다.

get used to 는 「～에 익숙하다. 」라는 의미 표현


(예)

get used to each other's habits

서로의 관습에 익숙해지다.




 



주의할점은


be used to 명사 <-> get used to명사의 차이점 


be used to +명사

-> [be동사]를 사용한경우: 익숙해 있다. 버릇이 되어있다로 상태를 나타낸다.


get used to 명사

-> [get]는 (  지금에서야 )익숙하게 되다.



 



부정문

be not used to ~ ing = ~하는 것이 익숙하지 않다.

be used to not ~ ing = ~하지 않는 것에 익숙하다.


부정문 예문:

I'm not used to dancing = 나는 춤추는게 익숙하지 않아.

I'm used to not dancing = 나는 춤추지 않는것이 익숙해 (=나는 평소에 춤을 안춰)




used 를 포함한 표현이 3가지 있습니다


주어 used to 동원 - 주어가 동원하곤 했다

주어 be used to 동원 - 주어가 사용되다 동원하는데

주어 be used to 명사/동명사 - 주어가 익숙하다 명사에/동명사하는 데에
