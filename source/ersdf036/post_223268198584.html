In order to와 so as to 다른점

In order to와 so as to는 모두 목적을 나타내는 구문이지만, 사용되는 상황과 문체에서 약간의 차이가 있습니다. 두 구문 모두 동사 앞에 사용되어 '어떤 일을 하기 위해' 또는 '어떤 결과를 달성하기 위해'라는 의미를 전달합니다. 그러나 그들 사이에는 몇 가지 미묘한 차이점이 있습니다.

​

In order to

보다 공식적이고 서술적인 어조: In order to는 보다 공식적인 문체에서 자주 사용됩니다.

학술 논문, 비즈니스 보고서, 공식적인 서신 등에서 자주 볼 수 있습니다.

간결함과 명확성: 이 구문은 문장에서 간결하고 명확한 목적을 제공합니다.

​

예시: She studied hard in order to pass the exam.

​

예문: In order to improve her English, she started reading novels in English.

해석: 그녀는 영어 실력을 향상시키기 위해 영어로 된 소설을 읽기 시작했다.

​

예문: He saved money in order to buy a new car.

해석: 그는 새 차를 사기 위해 돈을 저축했다.

​

예문: In order to complete the project on time, we must work extra hours.

해석: 프로젝트를 기한 내에 완성하기 위해서, 우리는 추가 시간을 일해야 한다.

​

So as to

​

보다 비공식적이거나 자연스러운 어조: So as to는 덜 공식적인 상황이나 구어체에서 자주 사용됩니다. 이

구문은 in order to보다는 좀 더 부드럽고 자연스러운 느낌을 줍니다.

때때로 더 복잡하거나 길게 느껴질 수 있음: 이 구문은 in order to보다 약간 더 길고 복잡하게 느껴질 수 있습니다.

따라서 간결함을 중시하는 문체에서는 덜 선호될 수 있습니다.

​

예시: He woke up early so as to catch the first train.

​

예문: She left early so as to avoid the traffic.

해석: 그녀는 교통 체증을 피하기 위해 일찍 출발했다.

​

예문: He whispered so as to not wake the baby.

해석: 그는 아기를 깨우지 않기 위해 속삭였다.

​

예문: They took the shortcut so as to arrive on time.

해석: 그들은 시간에 맞춰 도착하기 위해 지름길을 이용했다.

결론

In order to와 so as to는 기본적으로 같은 기능을 하지만,

in order to는 더 공식적이고 간결한 상황에, so as to는 덜 공식적이고 자연스러운 대화에 적합합니다.

어떤 구문을 사용할지는 문장의 맥락과 문체에 따라 결정되어야 합니다.
