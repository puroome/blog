1. Feel like~ing: ~하고싶다(그런기분이다)

This song is so good! I feel like dancing!

이 노래 정말 좋다! 춤 추고 싶어!

​

​

2. Feel like+명사: ~같이/~처럼 느낀다

​

반면 feel like의 like는 전치사입니다.

전치사인 만큼 like뒤엔 명사나 명사 상당어구가 오겠죠.

​

feel만 동사입니다.

'feel + like 명사' 로 쓰였을 경우에 like는 전치사로 쓰인 것이고

ex)

It’s perfectly normal to feel like this.

이렇게 느끼는 것은 지극히 정상적이다.

​

​

3. Feel like that절: (that절) 인 것 같이/처럼 느낀다

'feel + like S+V' 로 쓰였을 경우에 like는 접속사로 쓰인 것입니다.

ex)

I feel like that I’ve died and gone to heaven.

나는 죽어서 하늘나라에 간 기분이다.

​

​

예문) Patenting well feels like a moving target.

​

우선 해석해보면, "육아를 잘 하는 일은 움직이는 표적처럼 느껴진다."가 되는데

'움직이는 표적처럼 느껴진다'는 말은 의역하면

'그때 그때 상황에 맞추어 적절한 결정을 해야 되는 일'인 것 같다는 말입니다.

​

여기서 feel like는 feel like -ing 구문과는 다른 용법으로 쓰였습니다.

이해에 도움이 되도록 몇 가지 유사표현을 적어볼게요.

​

1. It feels like summer today. (오늘은 여름 같이 느껴진다. 즉 날씨가 덥다는 의미)

​

2. This place feels like home. Everything is so comfortable. (이곳은 완전 집 같아. 모든 게 편해)

​

3. He feels like a hero now. (그는 지금 영웅이 된 느낌이야.)

​

4. I feel like drinking coffee. (커피를 마시고 싶어.) = I want to drink coffee. 와 거의 같은 의미.

​

5. I don't feel like eating tonight. (난 오늘밤 먹고싶은 생각이 없어.)

​

feel like 다음에 동명사구가 올 때와 그냥 명사만 올 때 의미가 달라집니다.

^^ 하나는 '하고 싶다' 하나는 '~처럼 느껴진다'가 되죠.
