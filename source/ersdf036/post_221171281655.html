one/ones를 쓰는이유





one/ones를 쓰는이유에 대해서 이야기 해볼게요.



상황영어

these chocolates are nice

(이 초콜릿들 맛있어!)


would you like one?

(초콜릿 먹고싶니?)



초콜릿이라는 서로 이미 알고있는 단어를 계속

대화를 주고 받을때 one / ones를 사용합니다.


이유는 귀찮아서 입니다 ^^


1. 불필요한 반복을 피하기 위해서 입니다.


2. 서로 알고 있는 내용을 말할대 입니다.


one = a 와 같은 개념으로 통합니다.







서로 알고 있는 것에 대해서 대화를 할때

단수일때
주로 사용 되는 표현

this one / that one
the one / another one
the + 형용사 + one
이런 위주의 표현들을 영어회화 에서 자주 씁니다.


한국말에서도 대화하다가 
"아! 그거 그거! 그걸로 주세요.. 할때 그거!~"
[ the one ]

한국말에도 대화중에
"이걸로 줄까 저걸로 줄까?"
[ this one or that one? ]

이럴때 쓰인답니다.





앞에서 설명한것과 같은 내용입니다.

하지만 복수일때는 one에 s를 붙여서 복수로 표현할뿐
나머지는 의미하는게 똑같습니다.

 
서로 알고 있는 것에 대해서 대화를 할때

단수일때
주로 사용 되는 표현

these ones / those ones
the ones / the other ones
some + 형용사 + ones
