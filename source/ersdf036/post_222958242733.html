가정법 과거완료

If + S + had + p.p. ~, S + would/could/might/should + have p.p. ~.

'과거에 ~했더라면 ~(과거에)했을텐데' 일 때는 가정법 과거완료,

​

가정법 과거완료

EX)

If i hadn't watched ‘Cruella' , i wouldn't have been interested in fashion.

만약 내가 크루엘라를 보지 않았더라면 나는 패션에 관심을 가지지 못했을 것이다.

​

EX)

I regret If I had tried harder I could have better result.

내가 더 열심히 노력했다면 더 좋은 결과를 얻을 수 있었을 것을 후회한다.

​

혼합가정법

IF + S + had + p.p. S + would/could/might/should + R

If + S + 과거시제 동사 + would/could/might/should + have p.p.

'과거에 ~했더라면 ~(지금)할 텐데'이면 혼합가정법을 사용합니다.

​

[가정법과거]와 [가정법과거완료] 둘을 혼합해서 쓰는 것입니다

그럼 가정절은 가정법 과거로 하고 주절은 가정법과거완료의 혼합가정법으로 볼 순 있는데...

보통 가정절의 내용에 따라 주절이 결과적으로 나옵니다.

​

그래서 논리적으로 따지면 가정절의 사건이 주절의 사건보다 먼저 일어나서

​

이런 혼합가정법의 경우는 거의 다 가정절을 가정법과거완료로 하고 주절을 가정법과거로 합니다.

EX)

It was not clear how the banks would respond if their customers had already spent the money.
