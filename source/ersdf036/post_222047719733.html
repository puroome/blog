장소 부사: 말 그대로 장소를 나타내는 부사입니다.



장소 부사 특징:


*장소부사가 문두로 나와도 도치는 선택적으로 일어난다.

*주어가 대명사일 땐 장소부사가 문두로 나올때 도치가 발생하지 않는다 





For Example:

There was snow everywhere.

사방에 눈이 내렸다.


Shiny did not want to go there.

샤이니는 그곳에 가고 싶지 않았다.


위의 예에서 다음과 같은 단어들은 "장소부사"이다: 

everywhere(도처에,모든곳에 )there (거기에)

 



자주 쓰이는 장소부사 리스트:

here,there,home,abroad,outside,nowhere

anywhere,aside,elsewhere,upstairs,somewhere

underground,northwards,westwards,eastwards

southwards,upwards



장소부사는 전치사의 기능:

up,by,off,in,next,over,besides,across,under,behind,around





장소 부사: 보통 주동사 뒤에나 목적어 뒤에 배치


주동사 뒤 경우:

ex)

The cat looked away/up/down/around.

I'm going home/out/back.



목적어 뒤 경우:

ex)

They built a hut nearby.

He took the child outside.



 



장소부사의 도치방법:


1형식 문장에서...

장소부사(구)를 문두로 내보낼 때

주어와 동사를 도치시키되,

조동사 do를 사용하지 않습니다.



3형식 문장에서는...

장소부사(구)를 문두로 내보내더라도

주어와 동사를 도치시키지 않습니다.

In the room did I eat an apple. (x)

In the room I ate an apple. (o)



조동사도 동사의 일부라고 보기때문에

같이 도치하시면 된답니다 : )

항상 강조하고 싶은 부사구 / 부정어 / 명사를 도치시킬 때

조동사를 포함한 동사가 앞으로 나가게 된다는 점 기억해두셔요 : )




 



장소부사 도치방법:

주어+동사의 어순을 바꿉니다.

동사가 완전 자동사 일때 (be동사...)

본동사가 장소동사(sit, stand, live, lie etc,)나, 장소 이동동사(climb, come, go, walk, roll, run, rush, swim, fly.)일 때



 



장소부사 도치안되는 경우

진행형 시제일 때

주어가 대명사일 때

목적어가 있는 타동사

장소나 이동동사가 아닌 이외의 다른 동사일 때

타동사 뒤에 양태부사(slowly, happily, quickly 등)가 올 때
