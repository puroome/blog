Where이 접속사이면서 관계부사일 수 있나요?

Where there is a will, there is a way.

접속사인지 관계부사인지 명확히 구분해야하나요?


 



 우리말의 경우처럼

의미를 알고 바로 쓰면 되지

문법을 다 구분할 필요는 없습니다.



관계사는 

what과 복합관계대명사를 제외하고는


선행사가 있습니다.

선행사는 앞에 있는 동일한 대상을 말합니다.



 

그러므로

where there is a will

과 같이 문장의 처음에 나와 있다면

그 앞에 있는 선행사가 없습니다.

따라서 이 경우는 관계사가 아닙니다.

부사절 접속사이지요.


사실은 의미를 보아서

뜻이 있는 곳에 길이 있다

라는 의미로 where there is a will이 

부사적인 의미를 지니고 있다는 것


 


그리고

where there is a will을 빼도 

there is way가 완전한 문장이 되며

where there is a will이 수식하는 명사가 없으므로

이는 명사를 수식하는 것 (형용사절-관계절)이 아니라

동사나 문장을 수식하는 것 (부사절)임을 알 수 있습니다.
