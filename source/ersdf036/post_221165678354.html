each VS every 차이점 및 문법적설명




 


알쏭달송 똑같은느낌 하지만 다른 each VS every 차이점을 설명해볼게요.


each : (각각),전체 그룹에 속한 <== 개별적 사물 혹은 사람을 강조하고 싶을때 each를사용

every: (모두),전체를 강조하고 싶을때 사용함









 

문법적 사용스킬:


1.each는 대명사.형용사.부사로 쓰임 / every는 형용사로만 쓰임

2.each/every 다음 오는 명사는 반드시 단수형사용

3.each of / every one of 다음에는 복수명사 사용

            

each +      + 단수명사  + 단수동사


each of the + 복수명사  + 단수동사


each of     + 대명사    + 단수통사



every       + 단수명사  + 단수동사







each VS every 차이점을 이해하셨다면
연습문제를 풀어보자구요~
