“There is no -ing ~” 완벽 가이드

(불가능/통제 불가능/판단 불가를 자연스럽게 말하는 법)

​

1) 한 줄 정의

​

There is no + 동명사(-ing)

→ “~할 수 없다 / ~하는 건 불가능하다 / ~해봤자 소용없다”라는 강한 불가능·단정을 담은 고정표현이야.

​

There is no denying that… = “…라는 사실을 부정할 수 없다(= 누구도 부인 못함)”

​

There is no telling… = “~은 알 수 없다(= 예측 불가)”

​

There is no knowing whether… = “~인지 아닌지 알 수 없다”

​

핵심 포인트: 사람의 의지나 노력으로 바꾸기 힘든 상황, 미래 예측 불가, 논쟁의 여지 없는 사실을 말할 때 특히 잘 어울려.

​

2) 뉘앙스 디테일

​

**단순 불가능(cannot)**보다 담백하지만 단호해. 감정 과잉 없이 “그건 원천적으로 어려워요”라는 톤.

​

We cannot deny… 보다 There is no denying… 이 더 “부정의 여지 없음”을 강조.

​

**‘시도 자체가 무의미’**하다는 뉘앙스가 종종 깔림.

​

There is no use arguing. (말다툼해봐야 소용없다)

​

3) 기본 공식 & 자주 쓰는 동사

공식

​

There is no + 동명사(-ing) + (that/what/whether절 등 보어)

​

대표 콜로케이션(자연스럽게 붙는 동사들)

​

denying: There is no denying that he is talented.

​

telling / knowing / saying: There is no telling/knowing/saying what will happen.

​

use ~ing: There is no use crying over spilt milk. (엎질러진 우유는 울어도 소용없다)

​

point (in) ~ing: There is no point (in) arguing. (말다툼해봐야 의미 없다)

​

escaping/avoiding: There is no escaping the deadline. (마감은 피할 수 없다)

​

turning back: There is no turning back now. (이제 되돌릴 수 없다)

​

comparison: There is no comparing her vocals with anyone else’s. (비교 불가급)

​

주의: There is no to do는 틀림. 반드시 -ing 형태!

​

4) 문장 구조 확장하기

​

That절로 ‘사실’ 강조

​

There is no denying that prices are rising.

→ “물가가 오르는 건 부정할 수 없다.”

​

What/Whether절로 ‘정보 자체의 불확실성’

​

There is no telling what they will do next.

​

There is no knowing whether she will come.

→ “무엇을 할지/올지 말지 알 수 없다.”

​

전치사 in과의 결합 (use/point in)

​

There is no use in trying to contact him at midnight.

​

There is no point in blaming others.

→ in은 생략 가능하지만 넣으면 더 자연스러울 때가 많아.

​

5) 비슷한 표현 비교

​

You can’t tell / It’s impossible to know

​

말하는 사람의 능력/상황이 안 돼서 모른다는 뉘앙스.

​

There is no telling은 원천적으로 예측 자체가 불가하다는, 더 일반적·객관적 톤.

​

It’s no use ~ing / It’s no good ~ing

​

결과가 무익/무의미함을 강조. There is no use ~ing와 실무적으로 거의 동일 톤.

​

There is no such thing as ~

​

“~라는 건 존재하지 않는다.”(개념 부정) → 결이 다름.

​

There is no such thing as a free lunch.

​

6) 시제·조동사와 함께 쓰기

​

시제는 주로 주변 문맥으로 표현하고, -ing 자체는 형태가 변하지 않아.

​

There was no denying that she was right yesterday.

​

There will be no escaping the audit next month.

​

7) 실제 생활 예문 묶음

​

일정·업무

​

There is no avoiding the quarterly report; let’s start now.

​

There is no point in postponing the decision any longer.

​

공부·자기계발

​

There is no denying that daily practice works best.

​

There is no use cramming the night before the test.

​

인간관계·대화

​

There is no use arguing when neither side is listening.

​

There is no telling how he will react to the news.

​

금융·시장

​

There is no predicting short-term price swings with certainty.

​

There is no denying that the trend has weakened.

​

생활 조언

​

There is no turning back once you click “submit,” so review again.

​

There is no use crying over spilt milk; fix what you can now.

​

8) 자주 하는 실수

​

❌ There is no to deny / There is no deny → -ing 필수

​

❌ There is no telling about (대개 about 불필요)

​

❌ There is no the point in ~ (the 넣지 않음)

​

9) 한국어 → 영어 전환 팁

​

“알 수 없다/예측 불가” → There is no telling/knowing

​

“부정할 수 없다/명백하다” → There is no denying that ~

​

“소용없다/무익하다” → There is no use ~ing / There is no point (in) ~ing

​

“피할 수 없다/불가피” → There is no avoiding/escaping ~

​

10) 집중 예문 해부

A. There is no denying that he is talented.

​

의미: 그의 재능은 부정 불가(=명백)

​

포인트: 사실 판단에 대한 객관적 단정 분위기

​

B. There is no telling what will happen tomorrow.

​

의미: 내일 무슨 일이 일어날지 전혀 알 수 없음

​

포인트: 미래 예측 불가를 깔끔히 표현

​

C. There is no knowing whether she will come or not.

​

의미: 그녀가 올지 알 수 없음

포인트: whether절로 “A인지 아닌지” 구조를 자연스럽게 담기

​

11) 미니 대화

A: Should we buy tickets now or wait?

B: There is no telling what the prices will be tomorrow. Let’s lock them in today.

​

A: Can we avoid the compliance review?

B: There is no escaping it. We’d better prepare.
