​

so do I = 긍정문일때 사용합니다.

예를들어 I like cat 이런식으로 긍정문일때 자신도 그렇다면

so do I (나도 그래) 이런 식으로 사용합니다.

​

반면 neither do I= 상대가 먼저 부정문을 사용할때

자신도 그에 같은 입장인 상황에 씁니다.

예문으로 I don't like cat.

이렇게 부정문을 쓸때 자기도

그렇다면 Neither do I. ( 나도 고양이 싫어해. ) 이런 식으로 사용합니다.

​

예를 들어,

A: I like pasta. (나는 파스타를 좋아해.)

B: So do I. (나도 그래.)

​

A: I don't like spicy food. (나는 매운 음식을 싫어해.)

B: Neither do I. (나도 그래.)

​

이렇게 사용할 수 있습니다.

만약에 호응하지 않으려면, 반대되는 표현을 쓰면 됩니다.

​

예를 들어,

A: I like pasta. (나는 파스타를 좋아해.)

B: I don't. (난 안 그래.)

​

A: I don't like spicy food. (나는 매운 음식을 싫어해.)

B: I do. (난 그래.)

이렇게 사용할 수 있습니다.
