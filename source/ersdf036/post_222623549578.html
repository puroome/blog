as if 과거시제와 as if 현재시제의 구별하는 방법


She travels around the world as if money is no object. 

그녀는 돈은 신경 안 써도 되는 것처럼 전 세계를 돌아다닌다.


He behaves as if he doesn't know shameness at all. 

그는 수치스러움 이라고는 모르는 것처럼 행동한다.



as if절 현재시제는 직설법으로 '실현 가능한 경우' 를 표현



as if 과거 시제는  가정법으로 '현재와 반대의 사실을 가정






He talks as if he knew the answer.


He looks as if he didn’t sleep well.


​


첫번째 문장은 

주절 talk가 현재, if 부사절은 과거 -> as if 가정법 과거 형태의 문장 

[ 그는 정답을 아는 것처럼 말했다.. ( 화자는 he가 사실은 정답을 모를 거라고 생각하며 하는 말 )


여기서 의미상 knew는 talk와 시제가 같음 

즉 이야기 하는 시점과 그 정답을 알고 있는 시점이 의미상 일치함 ]



두번째 문장도 

주절이 현재, if 부사절은 과거 -> as if 가정법 과거 ( x ) 로

형태는 가정법 과거가 맞는데 가정법 과거로 쓸 수 없다고 함

저거는 무조건 as if 직설법으로 밖에 볼 수 없다고 함



즉, 문맥상 /논리상 구별합니다.

알지 못하고 아는 척하는 것이므로 논리상 맞으므로 현재의 반대를 가정하는 가정법과거
