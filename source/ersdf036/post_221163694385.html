It is~ there is~ 차이점 알아봅니다.






there is / are에 대해서 알아보기






1.첫번째:

there is / there are  = ~ 있다 / ~있어로 해석됩니다.


한국어로 풀이해보면

강남 사거라에 새로 생긴 맛집이 (있데/있어/있다)

이럴때 there is / there are를 씁니다.


there are 다음에 오는 식당이 복수이면 are

there are 다음에 오는 식당이 1개이면 is를 쓰는 간단한 원리 입니다.


ex)

there is an apple.  사과 하나가 있어.

there are apples.  사과 여러개가 있어


즉, there의 정의: 뭔가 우리가 이야기를 할때 (존재) 한다는 것을 말할때 주로 쓰임니다.



2.두번째:


there: 해석을 하면 "거기"에 라고 해석됩니다.

there는 이미 전치사 at/in/to가 포함되어 있습니다.

(home 역시 집에~라고 전치사가 포함되어 있어서 at/in/to를 사용하지 않는답니다.)


the new restaurant is very good.

I went there. = i went to the restaurant last night.

이처럼 there에 to전치사가 이미포함 되었다는 뜻.







IT에 대해서 알아보기


It is~ 는 특별히 해석하지 않습니다.

굳이 하자면 

"그것~  불라불라 해!"


영어문법에서 가주어라고 하는데

It이 내포하고 있는것은 뚜렷한 상황.사건.장소를 말합니다.


we went to the new restaurant.

(우리 새로운 식당 갔었어.)


It is very good.   It = the new restaurant.

(거기 매우 좋아)


there are many people in the restaurant.

그 식당에 사람 겁내 많아! (많이있어)
