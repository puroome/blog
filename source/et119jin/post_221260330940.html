clean은 닦으며 청소한다는 의미,

clean up은 어지러운 거 정리하라는 의미,

clean out은 지저분한걸 다 꺼내서 정리하라는 의미에요.

​

You need to clean the bathroom. (화장실 청소해라..화장실을 닦는 느낌)

You need to clean up your room. (방 치워...지저분한 거 치우는 느낌)

You need to clean out the garage. (차고 청소해 ... 지저분한 거 다 끌어내서 없애)

여기서는 clean up 써도 무방하답니다.
