Quiz.

1. A number of people (is, are, was, were) missing. 많은 사람들이 실종되었다.

2. The number of missing people (is, are) increasing. 실종된 사람의 수가 증가하고 있다.

3. The number of students (is, are) declining. 학생의 수가 감소하고 있다.

4. A number of girls (gets, get) doctor’s degree. 많은 여자들이 박사학위를 얻는다.

≫≫ Quiz 정답은 맨 아래쪽에 있어요~~!!

a number of 와 the number of를

구별하는 문제가

시험에 많이 나오는 이유가 뭘까요?

​

네.

관사때문이죠.

​

원래 의미를 가지고 있음에도

a, the로 인해

더 헷갈리게 되는거죠.

​

그래서 시험에 더 잘 나오는지도 모르죠.

​

그럼 의미를 살펴보면

주의할 점은

a number of의 주어는

of 뒤에 오는 복수명사가 주어가 되요.

​

a number of가 '많은'의 의미로

형용사처럼 뒤에 있는 명사를 꾸며주니

결국 복수취급을 해야 합니다.

​

하지만

the number of의 주어는

the number에요.

​

the number of가 '~의 수'이므로

of 뒤에 어떤 명사가 오든

명사가 주어가 아니에요.

그래서 시험에 잘 나온답니다.

두 개의 구분이 힘드시다면

제가 예전에 외웠던 방법인데

​

a(어) 많네

the(그) number(수)

외우셔도 되구요

예문을 살펴볼게요.

​

The number of teenagers using smart phone is on the rise.

스마트폰을 이용하는 십대들의 수는 증가하고 있다.

→ 주어는 the number of입니다.(teenagers가 아니에요)

→ the number of는 '~의 수'이고, 단수취급

​

A number of people are enjoying their afternoon in the park.

많은 사람들이 공원에서 그들의 오후를 즐기고 있다.

→ 주어는 people입니다.

→ a number of는 '많은~'의 의미고, 복수취급

​

The number of cars in Seoul is rapidly increasing.

서울의 자동차수가 급속히 늘고 있다.

→ 주어는 the number of입니다.(cars가 아니에요)

→ the number of는 '~의 수'이고, 단수취급

​

A number of people are opposing the land development.

많은 사람들은 토지 개발을 반대하고 있다.

→ 주어는 people입니다.

→ a number of는 '많은~'의 의미고, 복수취급

Quiz_Answer.

1. A number of people (is, are, was, were) missing. 많은 사람들이 실종되었다.

→ people이 주어, a number of(많은, 복수취급), 시제 과거

​

2. The number of missing people (is, are) increasing. 실종된 사람의 수가 증가하고 있다.

→ the number of가 주어(~의 수, 단수취급)

​

3. The number of students (is, are) declining. 학생의 수가 감소하고 있다.

→ the number of가 주어(~의 수, 단수취급)

​

4. A number of girls (gets, get) doctor’s degree. 많은 여자들이 박사학위를 얻는다.

→ girls가 주어, a number of(많은, 복수취급)
