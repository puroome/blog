아래의 문장이

맞는 문장인가요? 틀린 문장인가요?

틀렸다면 바르게 고쳐보세요.

​

​

Only five people did turn up.

오직 다섯 명의 사람들만 모습을 드러내었다.

​

​

​

어떤가요?

​

only가 문장의 맨 앞에 있고,

조동사 did + 동사원형 구조의

맞게 쓰인 문장이 맞나요?

​

​

​

정답을 알려드리면

위 문장은

문법적으로 틀린 문장입니다.

​

위 문장이 왜 틀렸을까요?

​

알아두셔야 할 것은

only 도치가 일어나는 경우는

아주 제한적이라는 것입니다.

​

즉,

only가 문장의 맨 앞에 위치할 때는

only + 부사(부사구, 부사절)일 경우에만

도치가 일어납니다.

​

​

​

하지만

Quiz의 문장은

only 뒤에 바로 주어가 나온 문장이죠.

​

Only five people turned up.

오직 다섯 명의 사람들만 모습을 드러내었다.

​

그럼

이 문장은 도대체 뭘까요?

​

'다른 누구도 아닌'

이해하셨나요?

​

​

​

지금부터

only 도치

(또는 only 부사구 도치)에 대해

아주 쉽게 풀어볼게요.

​

only 도치

(또는 only 부사구 도치)에는

3가지 형태가 있어요.

​

1) only + 부사

2) only + 부사구

3) only + 부사절

​

도치는

only가 문장의 맨 앞에 위치할 때

only + 부사(부사구, 부사절)일 경우에만

해당한다는 것을

꼭 기억하세요!!!

​

​

예문으로

확인해볼게요.

​

1) Only later did she realize her mistake.

​

2) Only in clear water does this fish live.

​

3) Only when he feels inclined to does he write.

​

이 문장들은

only 뒤에

부사, 부사구, 부사절이 와서

도치가 일어난 문장입니다.

​

부사를 표시를 해보면

​

1) Only later did she realize her mistake.(부사)

(← She realized her mistake only later.)

나중에서야 그녀는 자기 실수를 깨달았다.

​

2) Only in clear water does this fish live.(부사구)

(← This fish lives only in clear water.)

청정수역에서만 이 물고기가 산다.

​

3) Only when he feels inclined to does he write.(부사절)

(← He writes only when he feels inclined to.)

마음이 내킬 때에만 그는 글을 쓴다.

​

​

only가 문장의 맨 앞에 위치해서

only + 부사(부사구, 부사절)일 경우에만

도치가 일어난 문장입니다.

​

​

지금부터는

각각의 예문을

살펴볼게요.

​

1) only + 부사

​

Only then did I learn the unpalatable truth.

그제서야 나는 그 불쾌한 진실을 알게 되었다.

​

Only later did I understand what she meant.

나중에야 나는 그녀의 뜻을 이해했다.

​

Only then did she tell him about the attack.

그제서야 그녀는 그에게 그 공격에 대해 말했다.

​

​

2) only + 부사구

​

Only in Africa are thses flowers found.

이들 꽃은 아프리카에서만 발견된다.

​

Only in warm seas does the sea horse live.

따뜻한 바다에서만 해마가 산다.

​

Only in Paris can you see a restaurant like this.

이와 같은 식당은 파리에서만 볼 수 있다.

​

​

3) only + 부사절

​

Only when he needs something does he look for me.

그는 아쉬울 때만 나를 찾는다.

​

Only if the red light comes on is there any danger to employees.

빨간 불이 들어와 있을 때에만 직원들에게 무슨 위험이 있는 것이다.

​

Only if a teacher has given permission is a student allowed to leave the room.

학생은 교사의 허락이 있어야만 교실을 떠날 수 있다.

​

꼭 기억하세요!!!
