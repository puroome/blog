prosper

뜻 : 번창하다, 번영하다 (동사)

발음 : [prɑ: 프라- s 스 pə 퍼]

​

​

가게를 차렸는데, 장사가 잘 되서 돈을 많이 벌면 앞으로[pro] 가게를 더 늘릴까? 직원을 더 뽑을까? 고민하면서 희망[sper]을 갖게 되겠죠? 앞으로의 상황에 대해서 희망을 갖게 되는 것은 "번창하고, 번영하기" 때문이에요.

image from Pixabay / CC0

​

"prosper(번창하다, 번영하다)"는 [pro 앞에, 앞으로] + [sper 희망]으로 만들어지는 단어에요.

접두사(prefix)

어원(root)

접미사(suffix)

pro

sper

뜻 : 번영하는, 번창하는 (형용사)

​

"prosperous"(번영하는, 번창하는)"은 형용사 형태로 "prosper(번영하다, 번창하다)" + [ous 형용사]로 만들어져요.

단어

접미사(suffix)

prosper

ous

prosperous

prosperously

뜻 : 번영하면서, 번창하면서 (부사)

발음 : [prɑ: 프라- s 스 pə 퍼 rə 뤄 sli 슬리]

​

​

"prosperously"(번영하면서, 번창하면서)"는 부사 형태로 "prosperous(번영하는, 번창하는)" + [ly 부사]로 만들어져요.

단어

접미사(suffix)

prosperous

ly

prosperously

prosperity

뜻 : 번영, 번창 (명사)

발음 : [prɑ: 프라- s 스 pe 퍼 rə 뤄 ti 티]

​

​

"prosperity"(번영, 번창)"은 명사 형태로 "prosper(번영하다, 번창하다)" + [ty 명사]로 만들어져요.

단어

접미사(suffix)

prosper

ty

prosperity

단어가 만들어질 때 자연스러운 발음 연결을 위해서 모음 [i]가 추가 돼요.
