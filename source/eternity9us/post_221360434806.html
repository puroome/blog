connote
뜻 : 내포하다 (동사)
발음 : [kə 커 nóu 노우 t 트]

"뿌리가 약한 나무는 겨울을 넘지 못한다"라는 표현에는 "기본, 기반"이 약하면 어려움을 이겨낼 수 없다라는 "의미"를 함께[con] 알 수 있도록[not]해주는 표현이에요. 이런 것을 의미를 "내포한다"라고 해요.

"connote(내포하다)"는 [con 함께] + [not 알다]로 만들어져요.

단어가 만들어질 때 모음 [e]가 맨 뒤에 추가 돼요.

connotation
뜻 : 함축, 내포 (명사)
발음 : [kὰ 카 nə 너 téi 테이 ʃən 션]


"connotation(함축, 내포)"는 명사형태로 "connote(내포하다)" + [ate 동사] + [ion 명사]로 만들어지는 단어에요.

단어가 만들어질 때 발음상 불필요한 모음 [e]는 생략이 돼요.

denote
뜻 : 명확히 나타내다, 표시하다 (동사)
발음 : [di 디 nóu 노우 t 트]


"설탕을 어느 정도 넣어주세요"라는 불확실한 표현보다는 "설탕을 3g 넣어주세요"라는 표현이 완전히[de] 정확하게 알 수 있도록[not]해주는 표현이라고 할 수 있겠죠? 이런 것을 "명확히 나타내다, 표시하다"라고 해요.

"denote(명확히 나타내다, 표시하다)"는 [de 완전히] + [not 알다]로 만들어지는 단어에요. 


denotation
뜻 : 명시적 의미, 명확한 표시 (명사)
발음 : [dì: 디 nou 노우 téi 테이 ʃən 션]


"denotation(명시적 의미, 명확한 표시)"는 명사형태로 "denote(명확히 나타내다, 표시하다)" + [ate 동사] + [ion 명사]로 만들어져요.

단어가 만들어질 때 발음상 불필요한 모음 [e]는 생략이 돼요.
