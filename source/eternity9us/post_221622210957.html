code

뜻 : 암호, 특별한 규정 (명사) 코드화 하다 (동사)

발음 : [ko 코 u 우 d 드]

​

​

물건을 사서 계산대로 가져가면, 바코드(bar code)를 삑! 삑! 읽어서 자동으로 계산이 돼요. 바코드(bar code)는 막대기(bar) 모양으로 생긴 암호(code)의 일종으로 특별한 규정(code)에 따라서 그 의미가 지정되게 되어 있어요.

coding

뜻 : 암호화, 코딩 (명사)

발음 : [kóu 코우 diŋ 딩]

​

​

요즘은 학교에서도 컴퓨터의 명령어를 짜는 코딩(coding) 교육을 하고 있기 때문에 굉장히 익숙해진 단어에요. "coding(암호화, 코딩)"은 명사형태로 "code(암호, 특별한 규정, 코드화 하다)" + [ing 명사]로 만들어지는 단어에요.

단어가 만들어질 때 "code"에서 발음상 불필요한 모음 [e]는 생략 돼요.

decode

뜻 : 해독하다, 암호를 풀다 (동사)

발음 : [di: 디- ko 코 u 우 d 드]

​

​

"decode(해독하다, 암호를 풀다)"는 [de 반대] + "code(암호, 특별한 규정, 코드화 하다)"로 만들어지는 단어에요.

decoding

뜻 : 해독, 복호 (명사)

발음 : [di: 디- kóu 코우 diŋ 딩]

​

​

"decoding(해독, 복호)"는 명사형태로 "decode(해독하다, 암호를 풀다)" + [ing 명사]로 만들어지는 단어에요.

단어가 만들어질 때 "decode"에서 발음상 불필요한 모음 [e]는 생략 돼요.
