commutative

뜻 : <수학> 가환성의 (형용사)

발음 : [kά 카 mju 뮤 tèi 테이 ti 티 v 브]

[con 함께] + [mut] + [tive 형용사]

[com 함께] + [mut] + a + [tive 형용사]

​

"3-2 = 1"이라는 수식에서 두 숫자의 순서를 바꾸면 "2-3 = -1"처럼 결과가 바뀌게 돼요.

​

이와 달리 "2+3 = 5"라는 수식에서 두 숫자의 순서를 바꿔도 "3+2 = 5"처럼 결과 값은 같아요.

​

"commutative"는 이처럼 두 개의 순서를 서로 함께[con] 바꿔도[mut] 같은 결과가 나오는 상태[tive]라는 의미에서 "가환성의"라는 뜻이 돼요.
