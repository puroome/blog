이번에 학습할 단어는 "possible, possibly, possibility, impossible, impossibly, impossibility"라는 단어예요. 이 단어는 어원 [pot 힘]의 변화 형태인 [poss]에서 만들어져요.

​​

어원인 [pot 힘]과 관련된 설명 그리고 접두사, 접미사와 관련된 설명은 "어원과 접사 설명서"를 참고해주세요.

​* [pot 힘] - 어원 참고번호 106번

​​

​

possible

뜻 : 가능한 (형용사)

발음 : [pά 파 sə 서 bl 블]

[poss] + [ible 가능한]

​

충분한 힘[poss]이 있기 때문에 무언가를 해낼 수 있는[ible] 상태라는 의미에서 "가능한"이라는 뜻이 돼요.

possibly

뜻 : 어쩌면, 아마도, 어떻게 (부사)

발음 : [pά 파 sə 서 bli 블리]

possible + [ly 부사]

possible + [ly 부사]

​

얼마나 가능한(possible) 지를 생각해 보거나 의심해 본다는 의미에서 "어쩌면, 아마도, 어떻게"라는 뜻이 돼요.

possibility

뜻 : 가능성 (명사)

발음 : [pὰ 파 sə 서 bílə 빌러 ti 티]

possible + [ty 명사]

possibli + [ty 명사]

possibili + [ty 명사]

impossible

뜻 : 불가능한 (형용사)

발음 : [im 임 pά 파 sə 서 bl 블]

[in 반대] possible

[im 반대] possible

impossibly

뜻 : 불가능할 정도로 (부사)

발음 : [im 임 pά 파 sə 서 bli 블리]

impossible + [ly 부사]

impossible + [ly 부사]

impossibility

뜻 : 불가능, 불가능성 (명사)

발음 : [im 임 pὰ 파 sə 서 bílə 빌러 ti 티]

impossible + [ty 명사]

impossibli + [ty 명사]

impossibili + [ty 명사]
