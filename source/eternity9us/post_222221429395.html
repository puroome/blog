base

뜻 : 밑부분, 기초 (명사), 기초가 되다, 근거를 두다 (동사)

발음 : [bei 베이 s 스]

​

​

영어로 야구는 "baseball"이라고 불러요. 그 이유는 다이아몬드 모양의 꼭지점에 나무 판같은 것을 넣고 공을 던지고 치면서 이것을 밟고 들어오는 스포츠이기 때문이에요.

image from Pixabay / CC0

​

베이스(base)는 맨 아래에 놓이는 "밑부분"을 의미하기 때문에 "기초"라는 뜻도 될 수 있어요. 이 단어는 동사로 쓰이면 "기초가 되다" 또는 자신의 주장을 받쳐준다는 의미로 "근거를 두다"의 뜻으로 사용될 수 있어요.

basement

뜻 : 지하, 지하층 (명사)

발음 : [beis 베이스 mən 먼 t 트]

​

​

건물의 맨 밑에 놓여있는 "밑부분"은 지하층이 돼요. 엘리베이터를 탔을 때 B1, B2, B3... 이렇게 표시가 되는 것은 모두 basement(지하, 지하층)을 의미해요.

image from Pixabay / CC0

​

"basement(지하, 지하층)"은 명사 형태로 "base(밑부부, 기초, 기초가 되다,...)" + [ment 명사]로 만든어져요.

단어

접미사(suffix)

base

ment

basic

뜻 : 기초적인, 기본이 되는 (형용사)

발음 : [béi 베이 sik 직]

​

​

"basic(기초적인, 기본이 되는)"은 형용사 형태로 "base(밑부부, 기초, 기초가 되다,...)" + [ic 형용사]로 만든어져요.

단어

접미사(suffix)

ic

단어가 만들어질 때 "base"에서 발음상 불필요한 모음 [e]는 생략돼요.

basically

뜻 : 기초적으로, 기본적으로 (부사)

발음 : [béi 베이 si 지 kəli 컬리]

​

​

"basically(기초적으로, 기본적으로)"는 부사 형태로 "basic(기초적인, 기본이 되는)" + [al 형용사] + [ly 부사]로 만든어져요. 가운데 들어가는 [al]은 특별한 의미결합이라기 보다는 자연스러운 발음 연결을 위해서 들어간다고 할 수 있어요.

단어

접미사(suffix)

basic

al ly

basic

basis

뜻 : 근거 (명사)

발음 : [béi 베이 si 시 s 스]

​

​

"basis(근거)"는 명사 형태로 "base(근거가 되다,...)" + [sis 명사]로 만들어지는 단어예요.

단어

접미사(suffix)

base

sis

base

sis

basis

단어가 만들어질 때 "base"에서 발음상 겹쳐서 불필요한 철자 [se]는 생략돼요.

​

​

"base"와 "basis"는 둘 다 명사형태라서 혼동될 수도 있어요. "base"는 물리적인 기초에 사용되는 반면에 "basis" 논리의 "근거"처럼 개념적인 의미로 사용되는 단어예요.
