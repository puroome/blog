이번에 학습할 단어는 "humidify, humidification, humidifier"라는 단어예요. 이 단어들은 "humid(습기가 있는, 습한)"에서 이어지는 단어예요.

​​

"humid(습기가 있는, 습한)"와 관련된 설명은 아래 링크된 글을 참고해주세요.

​* humid(습기가 있는, 습한) 설명 보기

​​​

​

humidify

뜻 : 습하게 만들다, 축축하게 만들다 (동사)

발음 : [hju: 휴- mí 미 də 더 fài 파이]

humid + i + [fy 동사]

humidification

뜻 : 가습 (명사)

발음 : [hju 휴 mì 미 də 더 fi 피 kéi 케이 ʃən 션]

humidify + [ic 형용사] + [tion 명사]

humidify + [ic 형용사] + a + [tion 명사]

humidifier

뜻 : 가습기 (명사)

발음 : [hju: 휴- mí 미 də 더 fài 파이 ər 어]

humidify + [er ~것]

humidifi + [er ~것]
