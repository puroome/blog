implement

뜻 : 실행하다, 이행하다 (동사) 도구 (명사)

발음 : [ím 임 plə 플러 mən 먼 t 트]

​

​

Tom은 원하는 차를 사기위해서 3000만원을 모으기로 결심했어요. 그래서 Tom은 이것을 실행하려고 통장을 개설하고 매달 20만원씩 넣기 시작했어요. 이렇게 통장 안에[in] 돈을 채워넣는[ple] 것[ment]은 Tom이 목표한 것을 열심히 "실행하고, 이행하고"있다는 것을 나타낸다고 할 수 있겠죠?

​

"implement(실행하다, 이행하다, 도구)"는 [in 안에] + [ple 채우다] + [ment 명사]로 만들어지는 단어에요. 이 단어는 명사형 접미사가 붙어있지만, 기본적으로 동사의 의미로 사용돼요. "도구"라는 뜻는 무언가를 실행하거나 이행하기 위해서 사용하는 것이라는 의미에서 부차적으로 발생되는 뜻이에요.

​

"implement(도구)"라는 뜻을 이용해서 "stone implement(돌기구), farming implement(농기구), garden implement(원예 용구), writing implement(필기 도구)"와 같은 단어를 조합해서 사용할 수 있어요.

implementation

뜻 : 실행, 이행 (명사)

발음 : [ìm 임 plə 플러 mən 먼 téi 테이 ʃən 션]

​

​

"implementation(실행, 이행)"은 명사형태로 "implement(실행하다, 이행하다,..)" + [ate 동사] + [ion 명사]로 만들어지는 단어에요. 가운데 들어가는 [ate]는 특별한 의미결합이라기 보다는 자연스러운 발음 연결을 위해서 들어간다고 생각하면 돼요.
