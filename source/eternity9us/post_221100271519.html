direction
뜻 : 방향, 지휘, 감독 (명사)
발음 : [di 디 rek 뤡 ʃn 션 / daɪ 다이 rek 뤡 ʃn 션] 

"direct"는 직접적인, 똑바른 (형용사), 가리키다, 지시하다, 감독하다 (동사)의 의미를 가지고 있어요. direction은 direct(가리키다, 지시하다, 감독하다) + [ion 명사]로 만들어져요.

어딘가를 가리키는 것 - 방향
무언가를 지시하는 것 - 지휘
감독하는 것 - 감독


directory
뜻 : 주소록, 안내목록 (명사)
발음 : [dɪ 디 rek 뤡 tə 터 ri 리] 


복잡한 쇼핑몰 같은 곳을 가면, 가게들이 어디에 위치해 있는지 가리켜주는(direct) 안내판이 있죠? 이런 것을 영어로는 "directory"라고 부르는데, direct(가리키다....) + [ory 관련된 것]으로 만들어져요.
