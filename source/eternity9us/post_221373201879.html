accurate
뜻 : 정확한, 정밀한 (형용사)
발음 : [ǽ 애 kju 큐 rət 릿]

여러분들이 무언가를 틀리지 않고 해내려고 할 때, 또는 아주 작고 복잡한 무언가를 만들려고 할 때는 더욱 더 신경을 쓰면서 하곤 하죠? 이렇게 신경을 써서[cure] 해놓은 상태[ate]가 "정확한, 정밀한"이라는 뜻이 돼요.

"accurate(정확한, 정밀한)"은 [ad 더하다] + [cure 신경을 써주다, 돌보다] + [ate 형용사]로 만들어지는 단어에요. 신경쓰는 것을 더해서 즉, 신경을 많이 써서 해 놓은 상태를 의미해요.

단어가 만들어지면서 [ad]는 자연스러운 발음을 위해서 [ac]로 변화를 하고 [cure]에서 발음상 필요가 없는 모음 [e]는 생략이 돼요.

accurately
뜻 : 정확하게, 정밀하게 (부사)
발음 : [ǽ 애 kju 큐 rə 뤼 tli 틀리]


"accurately(정확하게, 정밀하게)"는 부사형태로 "accurate(정확한, 정밀한)" + [ly 부사]로 만들어지는 단어에요.


accuracy
뜻 : 정확함, 정밀함 (명사)
발음 : [ǽ 애 kju 큐 rə 뤄 si 시]


"accuracy(정확함, 정밀함)"은 명사형태로 "accurate(정확한, 정밀한)" + [cy 명사]로 만들어지는 단어에요.

단어가 만들어지면서 "accurate"에서 발음상 불필요한 [te]가 생략된다고 볼 수 있어요.
