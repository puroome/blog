circular
뜻 : 원의, 원형의 (명사)
발음 : [sə́:r 서 kjulər 큘러]

"circular(원의, 원형의)"는 "circle(원, 동그라미)" + [ar 형용사]로 만들어지는 단어에요.

단어가 만들어질 때 "circle"에서 발음에 필요없는 모음[e]는 생략이 돼요. 또한 보다 자연스러운 발음을 위해서 모음 [u]가 추가되는데, 이런 현상은 ~cle로 끝나는 단어에 [ar]이 붙을 때 공통적으로 나타나는 현상이에요.

circulate
뜻 : 순환시키다, 순환하다, 유통되다 (동사)
발음 : [sə́:r 서 kju 큘 lè 레 it 이트]


우리 몸속에서는 혈액이 온몸 구석 구석을 돌아다니면서 산소와 영양분을 공급해줘요. 온 몸을 돌아다닌  혈액은 다시 심장으로 돌아온 다음에 다시 온몸으로 뿜어져 나가는데, 마치 무한 반복되는 원(circle)처럼 도는 "순환"을 해요. 순환은 원과 같은 형태로 반복되는 것을 의미해요.

"circulate"는 "circle(원, 동그라미,...)" + [ate 동사]로 만들어지는 단어에요. 우리몸에서 피가 순환되는 것처럼 화폐나 재화가 시장에서 순환되는 것 - 즉, 유통되는 것도 마찬가지로 "circulate"라고 해요.


이 단어도 마찬가지로 단어가 만들어질 때 "circle"에서 발음에 필요없는 모음[e]는 생략이 돼요. 또한 보다 자연스러운 발음을 위해서 모음 [u]가 추가돼요.


circulation
뜻 : 순환, 유통 (명사)
발음 : [sə̀:r 서 kjuléi 큘레이 ʃən 션]


"circulation"은 명사형태로 "circulate(순환시키다, 순환하다, 유통되다)" + [ion 명사]로 만들어지는 단어에요.

단어가 만들어질 때 "circulate"에서 발음상 불 필요한 [e]는 생략이 돼요.
