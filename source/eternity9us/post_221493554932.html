layout

뜻 : 배치, 설계 (명사)

발음 : [lei 레이 a 아 ut 웃]

​

​

아래처럼 기계내부의 작동이 어떻게 되는지 기계 내부의 부품들이 서로 어떻게 연결 되어 있고 어떻게 놓여져(lay) 있는지 밖으로(out) 드러내 보이듯이 쉽게 알 수 있게 해주는 것이 바로 "배치, 설계"라고 해요.

layout image from Pixabay / CC0

​

"layout(배치, 설계)"는 "lay(놓다)" + "out(밖에, 밖으로)"로 만들어져요.

​

이 단어가 반드시 "기계"와 관련된 용어에만 해당되는 것은 아니에요. 아래처럼 홈페이지와 같은 것에서 각각의 구조를 어떤 식으로 배치를 하는지를 보여주는 것도 마찬가지로 "layout"이라고 해요.
