face

뜻 : 얼굴, 면 (명사) 마주하다, 직면하다 (동사)

발음 : [fe 페 is 이스]

​

​

우리에게 익숙한 Social media 중에는 "facebook"이라는 웹싸이트가 있어요. 우리는 이 싸이트를 통해서 온라인 상에서 다른 사람들과 마주할 수 있어요.

image from Pixabay / CC0

​

"얼굴(face)"이라고 하는 부분은 우리가 누구인지 밖으로 드러나는 "면"이기도 하고, 대외적으로 다른 사람들과 접촉할 때 드러내는 부분이기도해요. 그래서 동사로 사용되면 다른 사람또는 다른 무언가와 "마주하다, 직면하다"라는 뜻이 돼요.

surface

뜻 : 표면, 밖으로 드러난 층 (명사) 표면의 (형용사)

발음 : [​sə́:r 써- fi 피 s 스]

​

​

물리적인 특성을 가지는 대부분의 것들은 내부와 외부로 구분이 될 수 있어요. 중심이라고 생각할 수 있는 안쪽에서부터 다른 것들 이상으로[sur] - 즉, 다른 부분보다 더 멀리 떨어져 있는 부분이 바로 가장 바깥쪽이라고 할 수 있고, 이곳이 바로 외부와 접하는 면(face)을 가지게 되는데, 이러한 부분을 바로 "표면"이라고 해요.

image from Pixabay / CC0

​

"surface(표면, 밖으로 드러난 층)"은 [sur 초월한, ~이상의] + "face(얼굴, 면, 마주하다, 직면하다)"로 만들어지는 단어예요.

접두사(prefix)

단어

sur

face

surface

​

참고로 이 단어의 발음은 [써페이스]가 아니라 [써피스]처럼 발음이 되니깐 주의를 해야해요.

interface

뜻 : 접점, 접촉면, 컴퓨터의 인터페이스 (명사)

발음 : [​in 인 tər 터 fei 페이 s 스]

​

​

컴퓨터 용어에는 인터페이스(interface)라는 것이 있어요. 이것은 기기와 기기, 또는 기기와 사용자를 연결하는 "접점"을 의미하는 말로 사용자 인터페이스(UI : User Interface)라는 용어로도 많이 사용돼요.

image from Pixabay / CC0

​

우리는 이러한 인터페이스(interface)를 이용해서 직접 코드를 입력해서 명령을 내리지 않아도 간단한 클릭을 통해서 컴퓨터를 제어할 수도 있어요. 이렇게 둘 사이에서[inter] 서로 마주하는(face) 것이 가능하도록 해주는 장치 또는 부분을 "interface"라고 해요.

​

"interface(접점, 접촉면, 컴퓨터의 인터페이스)"는 [inter 상호간에, 서로간에] + "face(얼굴, 면, 마주하다, 직면하다)"로 만들어지는 단어예요.

접두사(prefix)

단어

inter

face

interface

bifacial

뜻 : 양면이 있는 (형용사)

발음 : [bai 바이 féi 페이 ʃəl 셜]

​

​

동전은 앞면과 뒷면처럼 두 개의[bi] 면(face)을 가지고 있죠? 이러한 상태를 "양면이 있는"이라고 해요.

​

"bifacial(양면이 있는)"는 [bi 둘] + "face(얼굴, 면,...)" + [al 형용사]로 만들어지는 단어예요.

단어(word)

접미사(suffix)
