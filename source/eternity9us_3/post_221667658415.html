tool

뜻 : 도구, 연장 (명사)

발음 : [tu:l 툴-]

​

​

"tool"은 특정한 작업을 할 때 사용하는 "도구, 연장"을 의미하는 단어에요.

image from Pixabay

​

컴퓨터를 활용하는데 도움이 주는 도구(tool)들도 있어요. 예를 들어 사진을 편리하게 볼 수 있도록 해 주는 프로그램, 화면을 캡쳐하는 프로그램, 파일을 전송하는 프로그램, 등등 여러 다양한 프로그램 도구(tool)를 제공해주는 싸이트 중에 알 툴즈(ALTools)라는 사이트가 있어요. 어째서 이런 이름을 지었는지 쉽게 알 수 있겠죠?

알툴즈 - 내 PC 필수품

알툴즈 제품 최신 소식을 가장 신속하게, 최신 버전 다운로드를 가장 간편하게 제공하는 알툴즈 공식 사이트입니다. 알툴즈 사이트에서 알툴즈 통합팩, 알집, 알씨, 알송, 알툴바, 알약, 알약 익스플로잇 쉴드, 알마인드, 알캡처, 알드라이브, 알키퍼 제품 정보를 확인하시고, 다운로드하세요!

www.altools.co.kr

toolbox

뜻 : 도구상자 (명사)

발음 : [tul 둘 bɔk 박 s 스]​

​

​

"toolbox"는 도구를 넣어두는 상자를 의미해요. 당연히 "tool(도구, 연장)" + "box(상자)"로 만들어지는 단어에요.

image from Pixabay

​

toolbar

뜻 : 툴바 (명사)

발음 : [tú:l 툴- bὰ:r 바]

​

​

"toolbar"는 인터넷 상단에 컴퓨터를 사용하면서 자주 사용하는 프로그램 도구(bar)를 배치해놓은 긴 막대기(bar)형태의 영역을 말하는 단어에요.

​

stool

뜻 : 걸상, 발판, 등받이가 없는 의자 (명사)

발음 : [s 스 tu:l 툴]

​

​

"stool"은 임시로 앉을 곳이 필요하거나 작업을 하기 위해서 조금 위치를 높여야 할 때 사용하는 등받이가 없는 의자를 의미하는 단어에요. 이 단어는 원래 "stand"에서 출발한 단어이지만, 보통 작업에서 또는 임시적인 목적으로 사용하는 올라서기 위한(stand) 도구(tool)로 이해를 하면 단어를 쉽게 기억해둘 수 있어요.
