1. computer illiterate - 컴맹


예문을 봐봅시다!







I'm computer-illiterate and sill afraid to learn how to use computers.

컴맹인 나는 아직도 컴퓨터 사용법을 배우는 게 두렵기만 하다.


Incomputerate people refer to someone computer-illiterate.

컴퓨터의 작동과 용어에 정통하지 않은 사람이란 컴퓨터 문맹을 지칭합니다.


* Incomputerate: 컴퓨터의 작동과 용어에 정통하지 않은

Guestrooms also feature refrigerators, complimentary high-speed wired Internet access, and air conditioning.

*complimentary:  무료의/ 호텔에서 손님에게 객실 및 식음료를 접대 및 판매촉진을 위해서 무료로 제공하는 것을 말합니다.

3. install - 설치하다


예문!







He installed a vaccine program

그는 바이러스 백신 프로그램을 설치했다. 


Install a printer using the Control Panel. 

[제어판]을 사용하여 프린터를 설치하십시오.







4. peripheral device - 주변기기(중앙 처리 장치와 대비해 카드 천공기·라인 프린터·자기 테이프 장치 등)


예문!







A graphics card is a peripherl device that enables the computer to process anddeliver graphics, animation, and videos.

그래픽 카드는 컴퓨터가 그래픽, 애니메이션 및 동영상을 처리하고 전달할 수 있도록 하는 주변 장치입니다.


컴퓨터에서 자주 볼 수 있는 표현!

Turn off all peripheral devices connected to this product. 

이 제품에 연결된 모든 주변 장치를 끄십시오.







5. connect / hook up - (전원·인터넷 등에) 연결하다 [연결시키다]


예문!





Check that the computer is hooked up to the printer.

컴퓨터가 프린터에 연결되어 있는지 확인하라.


Maybe it’s not hooked up right. 

아마 연결이 잘못됐나 봐요.
