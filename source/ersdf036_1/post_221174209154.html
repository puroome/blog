우체국 상황별 영어회화



1. Sending Mails

Helene: Hi. I’d like to send this by regular mail.

Ted: Do you already have a stamp? You need to buy one.


2. Sending Parcels

Victor: Do you send parcels abroad?

Wilfred: Yes we do. Please write down your personal information here.


3. Claiming Mails/Parcels

Isaac: Hi. I’m here for my package.

Sam: Do you have a delivery notification slip?

Isaac: Yes. Here it is.

Here are sample phrases that you can use when sending a mail or parcel

at a post office. Practice saying each. Can you add more examples on the

list?

• How much is a first class stamp?

• How much is a second class stamp?

• I’d like an envelope please.

• Could I have a book of first class

stamps please?

• I’d like to send this parcel to…

• I’d like to send this mail to…

• Where’s the post box?

• How much is the airmail to…

• I’d like to send this package to…

• Is there any mail for me?

• What’s inside the package?

• Do you have any boxes for parcels?

• Can I track it after it’s sent?

• I’ve come to collect a parcel.

• I’d like to send this through airmail.

• I’d like a jiffy bag please.

• How much will it cost to send this

parcel to…?

• Can you put it on the scales please?
