run around -을 바쁘게 돌다

 

He is running around the village.

그는 마을을 돌고 있어.

 

Stop running around me.

내 주변을 돌지 마라.

 

run at -에 돌진하다

 

A dog ran at the bird.

개 한 마리가 새를 향해 돌진했어.

 

A bus ran at the restaurant on the corner of the street.

버스 한 대가 거리의 한 구석에 있는 식당으로 돌진했다.

 

run by -에게 -을 말하다, 설명하다

 

Please run that by us again?

저희에게 다시 한 번 말씀해 주시겠어요?

 

run for -에 입후보하다

 

He ran for the Senate last year.

그는 작년에 상원의원에 출마했었어.

 

She is thinking of running for presidency.

그녀는 대통령 선거에 출마할 생각을 하고 있던데.

 

run into 딱 마주치다

 

I ran into my friend at the park.

공원에서 친구를 우연히 만났어.

 

Can you imagine whom I ran into?

너는 내가 누굴 만났는지 아니?

 

run on 계속하다

 

The movie ran on three hours.

영화가 3시간이나 계속되었어.

 

run out of -을 전부 사용하다

 

I'm running out of ideas.

이제 아이디어가 다 떨어져 가고 있어.

 

We've run out of sugar.

설탕이 떨어졌네.

 

run to -에 달려가다

 

All my family ran to the ocean.

우리 가족들은 모두 바다를 향해 달려갔다.

 

I'm going to run to you as soon as I finish my work.

일을 마치는 대로 너에게 달려갈게.

 

run up -을 늘리다, 쌓아가다

 

He ran up many bills.

그는 빚이 쌓여 있어요.

 

He ran up a lot of bills at the hotel.

그는 호텔에서 많은 금액을 못 냈어요.

 

run with -와 사귀다

 

I didn't know you ran with Susan.

나는 네가 수잔과 사귀는지 몰랐어.
