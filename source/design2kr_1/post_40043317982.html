start by 우선 -하다

 

Let's start by meeting with the department heads.

우선 부서장들과 회의를 먼저 시작하죠.

 

start for -로 출발하다

 

I'll start for Jeju Island next morning.

나는 내일 아침에 제주도로 떠날 거야.

 

When do you start for Lisbon?

넌 리스본으로 언제 떠나니?

 

start in on -에게 설교하다, -을 비판하다

 

Don't start in front of me.

나에게 설교하지 마.

 

While I was shopping, he kept starting in at me.

쇼핑을 하는 동안 그는 계속 나에게 잔소리를 하는 거 있지.

 

start on -에 착수하다

 

If you clean the table, I will start on a dessert.

네가 식탁을 정리해주면, 내가 디저트를 준비할게.

 

Let's start on washing the dishes.

설거지부터 하자.

 

start out 출발하다, 시작하다

 

We're going to start out at six.

우리는 6시에 출발할 계획이야.

 

start up -을 가동시키다. -을 창립하다

 

They started up two factories.

그들은 두 개의 공장을 가동시켰어요.

 

Why don't you start up the engine first?

엔진 시동부터 거는 게 어때?

 

start with -으로 시작하다

 

I'll start with fruits.

난 과일부터 먹을래.

 

You'd better start reading with this book.

이 책으로 읽기를 시작하는 게 좋을 거야.
