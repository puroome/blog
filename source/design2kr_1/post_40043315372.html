set by -을 -의 옆에 두다

 

You can set your umbrella by the door.

우산은 문 쪽에 두시면 돼요.

 

Don't set anything by the fire.

불 옆에 아무것도 두지 마라.

 

be set for -의 준비가 되어 있다

 

The soldiers were set for anything.

군인들은 어떤 것에도 준비가 되어 있었다.

 

You should set yourself up for every opportunity.

너는 가능한 모든 기회에 대비해야 해.

 

set in 시작되다

 

The panic set in when they saw the fire.

그들은 화재를 목격하고 패닉 상태가 되었다.

 

be set on -을 결심하고 있다

 

Jenny is set on her resignation, and there is nothing I can do.

제니는 사직을 결심했고, 저로서는 어쩔 도리가 없어요.

 

I am set on living abroad.

나는 해외에서 살기로 마음을 굳혔어.

 

set out 출발하다

 

He set out for Incheon at 9 o'clock.

그는 9시에 인천으로 출발했어.

 

When setting out  on a long walk, wear suitable shoes.

장거리 여행을 할 때는 적절한 신발을 신으세요.

 

set to -을 -에 맞추다

 

You set your watch to the local time.

네 시계를 지역 시간에 맞춰.

 

Do you know who set this value to the left?

누가 이 밸브를 왼쪽으로 고정시켰는지 아니?

 

set up -을 창립하다, -을 설정하다

 

He decided to set up a new company.

그는 새로운 회사를 만들기로 결심했다.

 

They agreed to set up a commission to investigate claims.

그들은 문제를 조사하기 위해 협약을 만드는 데 동의했다.
