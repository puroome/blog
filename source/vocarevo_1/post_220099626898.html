많은 문법책에서

some(약간, 몇몇, 좀)은 긍정문에 쓰고

any(어떤 ~라도, 어떤 것 ~라도)는

부정/의문문에 쓴다고 나와 있습니다.

그런 경향이 강한 건 사실입니다.

I want some of that.

나는 그것의 약간을 원한다.

(= 나는 그걸 좀 원한다.)

I don't want any of that.

나는 그것의 어떠한 것도 원하지 않는다.

(= 나는 그것을 전혀 원하지 않는다.)

Do you want any of that?

너는 그것의 어떠한 것이라도 원하니?

(= 너는 그거 조금이라도 원하니?)

 

그림 해석

There isn't any cheese.

어​떠한 치즈도 없어.

There aren't any bananas.

어떠한 바나나도 없어.

There isn't any jam.

어떠한 잼도 없어.

But there's some bread.

하지만 빵이 좀 있어.

Here, have some.

여기, 좀 먹어.

 

 

 

하지만...

뜻만 맞고 말만 되면

some을 부정/의문문에 쓸 수 있고

any도 긍정문에 쓸 수 있습니다.

그러니 두 단어의 정확한 뜻을 아는 것이

더 중요합니다.

 

some과 any의 뜻은...

some: 약간(의), 좀, 몇몇(의), 어떤

any: 어떤 (것) ~라도, 어떠한 (것)~라도

둘 다 가산명사/불가산 명사 구별 없이 쓸 수 있고

뒤에 명사를 붙여서 형용사처럼 쓸 수도 있고

단독으로 대명사처럼 쓸 수도 있어요.

 

I have some money.

긍정문 some

나는 약간의 돈을 갖고 있다. (돈 좀 있다)

Do you have some money?

의문문 some

너 돈 좀 있어?

Do you have some?

의문문 some

너 좀 (갖고) 있어?

 

 

I like some Americans.

긍정문 some

나는 몇몇 미국인들을 좋아한다.

I don't like some Americans.

부정문 some

나는 몇몇 미국인들을 안 좋아한다.

I don't like some.

부정문 some

나는 몇몇은 안 좋아한다.


 

I don't like any Americans.

부정문 any

나는 어떤 미국인들도 안 좋아한다.

I like any American food.

긍정문 any

나는 어떤 미국 음식이라도 좋아한다.

I like any.

긍정문 any

나는 어떤 것이라도 좋아한다.

(=나는 다 좋아한다).

 

 

Do you want some water?

의문문 some

물 좀 원하세요?

(= 물 좀 드릴까요?)

Do you want some?

의문문 some

좀 원하세요?

(= 좀 드릴까요?)

​Do you want any water?

의문문 any

어떠한 물이라도 원하세요?

(= 물 조금이라도 드릴까요? 혹시 물 원하세요?)

​Do you want any?

의문문 any

어떠한 것이라도 원하세요?

(= 조금이라도 드릴까요? 혹시 원하세요?)

 

 

Somebody didn't come.

긍정문 some(body)

누군가는 오지 않았다.

Anybody can come.

긍정문 any(body)

어떤 사람이라도 올 수 있다.

(= 아무나 와도 된다)
