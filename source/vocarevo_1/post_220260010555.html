so that... ...하도록​

​

그동안 so ~ that, such ~ that

구문들을 봤는데, 마지막으로

so that이 같이 붙어서 쓰이는 경우를

보겠습니다. ​


so that... ...하도록

행위와 그 행위를 하는 이유를

연결할 때 so (that)을 씁니다.

that은 언제든 생략할 수 있고요.



I'm not going to eat. 나는 먹지 않을 거야.

I'm not going to eat so (that) I can lose weight.

나는 먹지 않을 거야, [그래서] 살을 뺄 수 있도록.

*해석할 때 [그래서]를 넣어도 되고 안 넣어도 상관없어요.


​

I need money. 나는 돈이 필요해. 

​

I need money so (that) I can buy something to eat. 

나는 돈이 필요해, [그래서] 먹을 것을 살 수 있도록. 

​


I'm going to tell you. 나 너에게 말해줄게.

I'm going to tell you so (that) you know.

나 너에게 말해줄게,  [그래서] 니가 알도록.



He studied hard. 그는 열심히 공부했다.

He studied hard so (that) he wouldn't fail the test.

그는 열심히 공부했다, [그래서] 시험에 떨어지지 않도록.

*시제일치: study → will not fail, studied → would not fail




작문해 보세요.

나는 일자리가 필요하다, [그래서] 여자친구를 가질 수 있도록. 

I need a job so (that) I can have a girlfriend.


​

일어나, [그래서] 니가 학교 갈 수 있도록.

​

Get up so (that) you can go to school.
